# 第10章 图的基础

## 10.1 图的基本概念

### 图的定义与术语

图（Graph）是一种非线性数据结构，由顶点（Vertex）和边（Edge）组成。在数学中，图通常表示为 G = (V, E)，其中 V 是顶点的集合，E 是边的集合。

让我们先来看看图的一些基本术语：

- **顶点（Vertex/Node）**：图中的基本单元，也称为节点

- **边（Edge）**：连接两个顶点的线，表示顶点之间的关系

- **无向图（Undirected Graph）**：边没有方向，(A,B) 和 (B,A) 表示同一条边

- **有向图（Directed Graph/Digraph）**：边有方向，从一个顶点指向另一个顶点

- **权重（Weight）**：边上的数值，表示距离、成本等

- 度（Degree）

  ：与顶点相连的边的数量

  - 无向图中：度就是连接的边数
  - 有向图中：分为入度（指向该顶点的边数）和出度（从该顶点指出的边数）

- **路径（Path）**：从一个顶点到另一个顶点的顶点序列

- **环（Cycle）**：起点和终点相同的路径

- **连通图（Connected Graph）**：任意两个顶点之间都有路径的无向图

- **强连通图（Strongly Connected Graph）**：任意两个顶点之间都有双向路径的有向图

图在现实生活中有很多应用，比如：

- 社交网络（用户是顶点，好友关系是边）
- 地图导航（地点是顶点，道路是边）
- 网页链接（网页是顶点，超链接是边）
- 任务依赖关系（任务是顶点，依赖关系是边）

这节我们介绍了图的基本概念和术语，为后续学习图的存储和算法打下基础。

## 10.2 图的存储结构

### 图的存储方式概述

图的存储结构主要有两种经典方式：邻接矩阵和邻接表。选择哪种存储方式取决于图的特性和应用场景。

**邻接矩阵（Adjacency Matrix）**：

- 使用二维数组存储
- 适合稠密图（边数接近顶点数的平方）
- 空间复杂度：O(V²)，其中 V 是顶点数
- 查询两个顶点是否相邻的时间复杂度：O(1)

**邻接表（Adjacency List）**：

- 使用数组+链表（或其他动态结构）存储
- 适合稀疏图（边数远小于顶点数的平方）
- 空间复杂度：O(V + E)，其中 E 是边数
- 遍历某个顶点的所有邻接顶点的时间复杂度：O(度数)

还有其他存储方式如十字链表、邻接多重表等，但在实际应用中邻接矩阵和邻接表最为常用。

这节我们了解了图的主要存储结构及其适用场景，接下来我们将分别实现这两种存储方式。

## 10.3 邻接矩阵实现

### 邻接矩阵的数据结构

邻接矩阵使用二维数组来表示图中顶点之间的连接关系。对于无权图，矩阵元素通常用0和1表示是否连接；对于有权图，矩阵元素存储权重值。

#### 邻接矩阵实现方法表

| 功能名称     | 方法调用                                        | 具体功能与注意事项                                |
| ------------ | ----------------------------------------------- | ------------------------------------------------- |
| 初始化图     | initGraph(Graph* g, int vertexCount)            | 创建指定顶点数的图，初始化邻接矩阵                |
| 添加边       | addEdge(Graph* g, int from, int to, int weight) | 添加从from到to的边，weight为权重（无权图可设为1） |
| 删除边       | removeEdge(Graph* g, int from, int to)          | 删除指定的边，将对应矩阵元素设为0或无穷大         |
| 检查边存在   | hasEdge(Graph* g, int from, int to)             | 检查两个顶点之间是否存在边                        |
| 获取顶点度数 | getDegree(Graph* g, int vertex)                 | 计算指定顶点的度数（无向图）或出度（有向图）      |

下面是一个完整的邻接矩阵实现示例：

```c
#include <stdio.h>
#include <stdlib.h>
#include <stdbool.h>

// 定义无穷大常量，表示不存在的边
#define INF 999999
#define MAX_VERTICES 100

// 图的结构体定义
typedef struct {
    int vertices;           // 顶点数量
    int edges;              // 边数量
    bool isDirected;        // 是否为有向图
    int adjMatrix[MAX_VERTICES][MAX_VERTICES]; // 邻接矩阵
} Graph;

// 初始化图
// 参数：g - 指向图结构的指针，vertexCount - 顶点数量，directed - 是否为有向图
void initGraph(Graph* g, int vertexCount, bool directed) {
    if (vertexCount <= 0 || vertexCount > MAX_VERTICES) {
        printf("错误：顶点数量必须在1到%d之间\n", MAX_VERTICES);
        return;
    }
    
    g->vertices = vertexCount;
    g->edges = 0;
    g->isDirected = directed;
    
    // 初始化邻接矩阵，所有元素设为INF（表示无边）
    for (int i = 0; i < vertexCount; i++) {
        for (int j = 0; j < vertexCount; j++) {
            g->adjMatrix[i][j] = INF;
        }
        // 顶点到自身的距离设为0
        g->adjMatrix[i][i] = 0;
    }
}

// 添加边
// 参数：g - 指向图结构的指针，from - 起始顶点，to - 目标顶点，weight - 边的权重
void addEdge(Graph* g, int from, int to, int weight) {
    // 检查顶点索引是否有效
    if (from < 0 || from >= g->vertices || to < 0 || to >= g->vertices) {
        printf("错误：顶点索引超出范围\n");
        return;
    }
    
    // 设置边的权重
    g->adjMatrix[from][to] = weight;
    
    // 如果是无向图，需要同时设置反向边
    if (!g->isDirected) {
        g->adjMatrix[to][from] = weight;
    }
    
    g->edges++;
}

// 删除边
// 参数：g - 指向图结构的指针，from - 起始顶点，to - 目标顶点
void removeEdge(Graph* g, int from, int to) {
    // 检查顶点索引是否有效
    if (from < 0 || from >= g->vertices || to < 0 || to >= g->vertices) {
        printf("错误：顶点索引超出范围\n");
        return;
    }
    
    // 检查边是否存在
    if (g->adjMatrix[from][to] == INF) {
        printf("警告：边(%d, %d)不存在\n", from, to);
        return;
    }
    
    // 删除边
    g->adjMatrix[from][to] = INF;
    
    // 如果是无向图，同时删除反向边
    if (!g->isDirected && g->adjMatrix[to][from] != INF) {
        g->adjMatrix[to][from] = INF;
        g->edges--; // 无向图中一条边被计算两次，所以只减一次
    }
    
    g->edges--;
}

// 检查两个顶点之间是否存在边
// 参数：g - 指向图结构的指针，from - 起始顶点，to - 目标顶点
// 返回值：true表示存在边，false表示不存在
bool hasEdge(Graph* g, int from, int to) {
    // 检查顶点索引是否有效
    if (from < 0 || from >= g->vertices || to < 0 || to >= g->vertices) {
        return false;
    }
    
    return g->adjMatrix[from][to] != INF;
}

// 获取顶点的度数（无向图）或出度（有向图）
// 参数：g - 指向图结构的指针，vertex - 顶点索引
// 返回值：度数或出度
int getDegree(Graph* g, int vertex) {
    // 检查顶点索引是否有效
    if (vertex < 0 || vertex >= g->vertices) {
        printf("错误：顶点索引超出范围\n");
        return -1;
    }
    
    int degree = 0;
    for (int i = 0; i < g->vertices; i++) {
        if (g->adjMatrix[vertex][i] != INF && vertex != i) {
            degree++;
        }
    }
    return degree;
}

// 打印图的邻接矩阵
// 参数：g - 指向图结构的指针
void printGraph(Graph* g) {
    printf("图的邻接矩阵表示：\n");
    printf("  ");
    for (int i = 0; i < g->vertices; i++) {
        printf("%3d", i);
    }
    printf("\n");
    
    for (int i = 0; i < g->vertices; i++) {
        printf("%2d", i);
        for (int j = 0; j < g->vertices; j++) {
            if (g->adjMatrix[i][j] == INF) {
                printf("  ∞"); // 用∞表示无穷大
            } else {
                printf("%3d", g->adjMatrix[i][j]);
            }
        }
        printf("\n");
    }
    
    printf("顶点数: %d, 边数: %d, 有向图: %s\n", 
           g->vertices, g->edges, g->isDirected ? "是" : "否");
}

// 主函数 - 演示邻接矩阵的使用
int main() {
    // 创建一个有4个顶点的无向图
    Graph graph;
    initGraph(&graph, 4, false);
    
    // 添加边
    addEdge(&graph, 0, 1, 5);
    addEdge(&graph, 0, 2, 3);
    addEdge(&graph, 1, 2, 2);
    addEdge(&graph, 1, 3, 6);
    addEdge(&graph, 2, 3, 7);
    
    // 打印图
    printGraph(&graph);
    
    // 测试边的存在性
    printf("\n边(0,1)存在: %s\n", hasEdge(&graph, 0, 1) ? "是" : "否");
    printf("边(0,3)存在: %s\n", hasEdge(&graph, 0, 3) ? "是" : "否");
    
    // 获取顶点度数
    printf("\n顶点0的度数: %d\n", getDegree(&graph, 0));
    printf("顶点1的度数: %d\n", getDegree(&graph, 1));
    
    // 删除一条边
    printf("\n删除边(1,2)...\n");
    removeEdge(&graph, 1, 2);
    printGraph(&graph);
    
    return 0;
}
```

**注意事项：**

- 邻接矩阵的空间复杂度固定为O(V²)，即使图很稀疏也会占用大量空间
- 对于无权图，可以用1表示存在边，0表示不存在边
- 对于有权图，通常用特殊值（如INF）表示不存在的边
- 顶点通常用0到V-1的整数编号，便于数组索引
- 在实际应用中，可能需要根据具体需求调整INF的值

这节我们详细实现了邻接矩阵的图存储结构，包括基本操作和完整的示例代码。

## 10.4 邻接表实现

### 邻接表的数据结构

邻接表使用数组存储顶点，每个顶点对应一个链表，链表中存储与该顶点相邻的所有顶点。这种方式在存储稀疏图时比邻接矩阵更节省空间。

#### 邻接表实现方法表

| 功能名称     | 方法调用                                        | 具体功能与注意事项                   |
| ------------ | ----------------------------------------------- | ------------------------------------ |
| 初始化图     | initGraph(Graph* g, int vertexCount)            | 创建指定顶点数的图，初始化邻接表数组 |
| 添加边       | addEdge(Graph* g, int from, int to, int weight) | 在from顶点的链表中添加to顶点         |
| 删除边       | removeEdge(Graph* g, int from, int to)          | 从from顶点的链表中删除to顶点         |
| 查找邻接顶点 | findAdjacent(Graph* g, int vertex)              | 返回指定顶点的所有邻接顶点           |
| 释放图内存   | freeGraph(Graph* g)                             | 释放所有动态分配的内存               |

下面是邻接表的完整实现：

```c
#include <stdio.h>
#include <stdlib.h>
#include <stdbool.h>

// 定义边节点结构
typedef struct EdgeNode {
    int dest;           // 目标顶点
    int weight;         // 边的权重
    struct EdgeNode* next; // 指向下一个边节点
} EdgeNode;

// 定义图结构
typedef struct {
    int vertices;       // 顶点数量
    int edges;          // 边数量
    bool isDirected;    // 是否为有向图
    EdgeNode** adjList; // 邻接表数组，每个元素指向一个边链表
} Graph;

// 创建新的边节点
// 参数：dest - 目标顶点，weight - 边的权重
// 返回值：指向新创建边节点的指针
EdgeNode* createEdgeNode(int dest, int weight) {
    EdgeNode* newNode = (EdgeNode*)malloc(sizeof(EdgeNode));
    if (newNode == NULL) {
        printf("错误：内存分配失败\n");
        return NULL;
    }
    newNode->dest = dest;
    newNode->weight = weight;
    newNode->next = NULL;
    return newNode;
}

// 初始化图
// 参数：g - 指向图结构的指针，vertexCount - 顶点数量，directed - 是否为有向图
void initGraph(Graph* g, int vertexCount, bool directed) {
    if (vertexCount <= 0) {
        printf("错误：顶点数量必须大于0\n");
        return;
    }
    
    g->vertices = vertexCount;
    g->edges = 0;
    g->isDirected = directed;
    
    // 分配邻接表数组
    g->adjList = (EdgeNode**)malloc(vertexCount * sizeof(EdgeNode*));
    if (g->adjList == NULL) {
        printf("错误：内存分配失败\n");
        return;
    }
    
    // 初始化每个顶点的链表头为NULL
    for (int i = 0; i < vertexCount; i++) {
        g->adjList[i] = NULL;
    }
}

// 添加边
// 参数：g - 指向图结构的指针，from - 起始顶点，to - 目标顶点，weight - 边的权重
void addEdge(Graph* g, int from, int to, int weight) {
    // 检查顶点索引是否有效
    if (from < 0 || from >= g->vertices || to < 0 || to >= g->vertices) {
        printf("错误：顶点索引超出范围\n");
        return;
    }
    
    // 创建新的边节点并插入到from顶点的链表头部
    EdgeNode* newNode = createEdgeNode(to, weight);
    if (newNode == NULL) {
        return; // 内存分配失败
    }
    newNode->next = g->adjList[from];
    g->adjList[from] = newNode;
    
    // 如果是无向图，还需要添加反向边
    if (!g->isDirected) {
        EdgeNode* reverseNode = createEdgeNode(from, weight);
        if (reverseNode == NULL) {
            // 注意：这里存在内存泄漏风险，实际项目中需要更好的错误处理
            return;
        }
        reverseNode->next = g->adjList[to];
        g->adjList[to] = reverseNode;
    }
    
    g->edges++;
}

// 在链表中查找并删除指定目标顶点的边
// 参数：head - 链表头指针的地址，dest - 要删除的目标顶点
// 返回值：true表示成功删除，false表示未找到
bool removeEdgeFromList(EdgeNode** head, int dest) {
    if (*head == NULL) {
        return false;
    }
    
    // 如果要删除的是头节点
    if ((*head)->dest == dest) {
        EdgeNode* temp = *head;
        *head = (*head)->next;
        free(temp);
        return true;
    }
    
    // 在链表中查找要删除的节点
    EdgeNode* current = *head;
    while (current->next != NULL && current->next->dest != dest) {
        current = current->next;
    }
    
    // 如果找到了要删除的节点
    if (current->next != NULL) {
        EdgeNode* temp = current->next;
        current->next = current->next->next;
        free(temp);
        return true;
    }
    
    return false; // 未找到
}

// 删除边
// 参数：g - 指向图结构的指针，from - 起始顶点，to - 目标顶点
void removeEdge(Graph* g, int from, int to) {
    // 检查顶点索引是否有效
    if (from < 0 || from >= g->vertices || to < 0 || to >= g->vertices) {
        printf("错误：顶点索引超出范围\n");
        return;
    }
    
    // 尝试从from顶点的链表中删除to
    if (removeEdgeFromList(&(g->adjList[from]), to)) {
        g->edges--;
    } else {
        printf("警告：边(%d, %d)不存在\n", from, to);
        return;
    }
    
    // 如果是无向图，还需要删除反向边
    if (!g->isDirected) {
        removeEdgeFromList(&(g->adjList[to]), from);
        // 注意：无向图中边被计算两次，但我们只减一次edges
    }
}

// 打印图的邻接表表示
// 参数：g - 指向图结构的指针
void printGraph(Graph* g) {
    printf("图的邻接表表示：\n");
    for (int i = 0; i < g->vertices; i++) {
        printf("顶点 %d: ", i);
        EdgeNode* current = g->adjList[i];
        if (current == NULL) {
            printf("无邻接顶点\n");
        } else {
            while (current != NULL) {
                printf("(%d, %d) ", current->dest, current->weight);
                current = current->next;
            }
            printf("\n");
        }
    }
    
    printf("顶点数: %d, 边数: %d, 有向图: %s\n", 
           g->vertices, g->edges, g->isDirected ? "是" : "否");
}

// 获取顶点的度数（无向图）或出度（有向图）
// 参数：g - 指向图结构的指针，vertex - 顶点索引
// 返回值：度数或出度
int getDegree(Graph* g, int vertex) {
    // 检查顶点索引是否有效
    if (vertex < 0 || vertex >= g->vertices) {
        printf("错误：顶点索引超出范围\n");
        return -1;
    }
    
    int degree = 0;
    EdgeNode* current = g->adjList[vertex];
    while (current != NULL) {
        degree++;
        current = current->next;
    }
    return degree;
}

// 释放图的所有内存
// 参数：g - 指向图结构的指针
void freeGraph(Graph* g) {
    for (int i = 0; i < g->vertices; i++) {
        EdgeNode* current = g->adjList[i];
        while (current != NULL) {
            EdgeNode* temp = current;
            current = current->next;
            free(temp);
        }
    }
    free(g->adjList);
    g->adjList = NULL;
    g->vertices = 0;
    g->edges = 0;
}

// 主函数 - 演示邻接表的使用
int main() {
    // 创建一个有4个顶点的无向图
    Graph graph;
    initGraph(&graph, 4, false);
    
    // 添加边
    addEdge(&graph, 0, 1, 5);
    addEdge(&graph, 0, 2, 3);
    addEdge(&graph, 1, 2, 2);
    addEdge(&graph, 1, 3, 6);
    addEdge(&graph, 2, 3, 7);
    
    // 打印图
    printGraph(&graph);
    
    // 获取顶点度数
    printf("\n顶点0的度数: %d\n", getDegree(&graph, 0));
    printf("顶点1的度数: %d\n", getDegree(&graph, 1));
    
    // 删除一条边
    printf("\n删除边(1,2)...\n");
    removeEdge(&graph, 1, 2);
    printGraph(&graph);
    
    // 释放内存
    freeGraph(&graph);
    
    return 0;
}
```

**注意事项：**

- 邻接表使用动态内存分配，需要手动管理内存，避免内存泄漏
- 插入边时通常采用头插法，时间复杂度为O(1)
- 删除边需要遍历链表，时间复杂度为O(度数)
- 对于无向图，每条边需要存储两次，增加了存储开销但简化了操作
- 邻接表的空间复杂度为O(V + E)，比邻接矩阵更适合稀疏图

这节我们实现了邻接表的图存储结构，包括内存管理和完整的操作函数。
