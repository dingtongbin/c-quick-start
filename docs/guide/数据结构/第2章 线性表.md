# 第2章 线性表

## 2.1 线性表基本概念

线性表是最基础也是最重要的数据结构之一。你可以把它想象成一排整齐站好的士兵，每个人都知道自己前面和后面是谁。在计算机中，线性表就是n个数据元素的有限序列，这些元素具有相同的数据类型，并且相邻元素之间存在一对一的关系。

### 线性表基本操作方法表

| 功能名称 | 函数原型                                       | 具体功能与注意事项                 |
| -------- | ---------------------------------------------- | ---------------------------------- |
| 初始化   | `void initList(List *list)`                    | 创建空的线性表，需要传入线性表指针 |
| 销毁     | `void destroyList(List *list)`                 | 释放线性表占用的内存空间           |
| 判空     | `bool isEmpty(List *list)`                     | 判断线性表是否为空，返回true/false |
| 求长度   | `int length(List *list)`                       | 返回线性表中元素的个数             |
| 获取元素 | `ElementType getElem(List *list, int index)`   | 根据位置获取元素，注意索引从1开始  |
| 查找元素 | `int locateElem(List *list, ElementType elem)` | 查找指定元素的位置，不存在返回0    |

### 使用示例

```c
#include <stdio.h>
#include <stdlib.h>
#include <stdbool.h>

// 定义元素类型
typedef int ElementType;

// 线性表结构定义（这里先用数组方式简单定义）
#define MAXSIZE 100
typedef struct {
    ElementType data[MAXSIZE];  // 存储数据的数组
    int length;                 // 当前长度
} List;

// 初始化线性表
void initList(List *list) {
    // 将长度初始化为0，表示空表
    list->length = 0;
}

// 判断线性表是否为空
bool isEmpty(List *list) {
    // 如果长度为0，则为空
    return list->length == 0;
}

// 获取线性表长度
int length(List *list) {
    // 直接返回当前长度
    return list->length;
}

// 在指定位置插入元素
bool insertElem(List *list, int index, ElementType elem) {
    // 检查插入位置是否合法（1到length+1之间）
    if (index < 1 || index > list->length + 1) {
        printf("插入位置不合法！\n");
        return false;
    }
    
    // 检查是否已满
    if (list->length >= MAXSIZE) {
        printf("线性表已满，无法插入！\n");
        return false;
    }
    
    // 将插入位置及后面的元素后移一位
    for (int i = list->length; i >= index; i--) {
        list->data[i] = list->data[i - 1];
    }
    
    // 在指定位置插入新元素
    list->data[index - 1] = elem;
    
    // 长度加1
    list->length++;
    
    return true;
}

// 删除指定位置的元素
bool deleteElem(List *list, int index, ElementType *elem) {
    // 检查删除位置是否合法（1到length之间）
    if (index < 1 || index > list->length) {
        printf("删除位置不合法！\n");
        return false;
    }
    
    // 保存要删除的元素
    *elem = list->data[index - 1];
    
    // 将删除位置后面的元素前移一位
    for (int i = index - 1; i < list->length - 1; i++) {
        list->data[i] = list->data[i + 1];
    }
    
    // 长度减1
    list->length--;
    
    return true;
}

// 主函数演示
int main() {
    List myList;
    
    // 初始化线性表
    initList(&myList);
    printf("初始化后，线性表是否为空：%s\n", isEmpty(&myList) ? "是" : "否");
    
    // 插入几个元素
    insertElem(&myList, 1, 10);  // 在位置1插入10
    insertElem(&myList, 2, 20);  // 在位置2插入20
    insertElem(&myList, 3, 30);  // 在位置3插入30
    
    printf("插入元素后，线性表长度：%d\n", length(&myList));
    
    // 删除元素
    ElementType deletedElem;
    if (deleteElem(&myList, 2, &deletedElem)) {
        printf("删除的元素是：%d\n", deletedElem);
        printf("删除后，线性表长度：%d\n", length(&myList));
    }
    
    return 0;
}
```

### 注意事项

1. **索引约定**：线性表的位置通常从1开始计数，而C语言数组从0开始，使用时要注意转换
2. **边界检查**：所有操作都要进行边界检查，避免数组越界访问
3. **内存管理**：如果是动态分配内存的线性表，要注意及时释放内存防止内存泄漏
4. **返回值处理**：插入、删除等操作可能失败，要检查返回值并做相应处理

线性表作为最基础的数据结构，为我们后续学习更复杂的数据结构打下了坚实的基础。理解线性表的基本概念和操作是学习数据结构的第一步，也是至关重要的一步。
