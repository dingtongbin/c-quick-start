# C语言数据结构教程

## 第3章 栈

### 3.1 栈的基本概念

栈是一种特殊的线性表，它只允许在表的一端进行插入和删除操作。这一端被称为栈顶（top），另一端被称为栈底（bottom）。栈遵循"后进先出"（Last In First Out，LIFO）的原则，就像一摞盘子，你只能从最上面拿盘子或者放盘子。

栈的基本操作包括：

- 初始化栈
- 入栈（push）：将元素添加到栈顶
- 出栈（pop）：移除并返回栈顶元素
- 获取栈顶元素（peek/top）：查看栈顶元素但不移除
- 判断栈是否为空
- 获取栈的大小

栈在计算机科学中有广泛的应用，比如函数调用、表达式求值、括号匹配、浏览器的后退功能等。

栈作为一种抽象数据类型，可以用数组（顺序存储）或链表（链式存储）来实现。

### 3.2 顺序栈实现

顺序栈是使用数组来实现的栈。我们需要维护一个指向栈顶的指针（通常是数组下标），以及栈的最大容量。

#### 顺序栈实现方法表

| 功能名称 | 实例调用方法                       | 具体功能与注意事项                              |
| -------- | ---------------------------------- | ----------------------------------------------- |
| 初始化栈 | `initStack(Stack *s, int maxSize)` | 创建一个最大容量为maxSize的空栈，top初始化为-1  |
| 入栈操作 | `push(Stack *s, int value)`        | 将value压入栈顶，需先检查栈是否已满             |
| 出栈操作 | `pop(Stack *s, int *value)`        | 弹出栈顶元素并保存到value中，需先检查栈是否为空 |
| 获取栈顶 | `peek(Stack *s, int *value)`       | 获取栈顶元素但不弹出，需先检查栈是否为空        |
| 判断空栈 | `isEmpty(Stack *s)`                | 检查栈是否为空，当top==-1时栈为空               |
| 判断满栈 | `isFull(Stack *s)`                 | 检查栈是否已满，当top==maxSize-1时栈满          |
| 获取大小 | `size(Stack *s)`                   | 返回当前栈中元素个数，即top+1                   |

下面是顺序栈的完整实现：

```c
#include <stdio.h>
#include <stdlib.h>
#include <stdbool.h>

// 定义栈结构体
typedef struct {
    int *data;      // 指向动态分配的数组
    int top;        // 栈顶指针，-1表示空栈
    int maxSize;    // 栈的最大容量
} Stack;

// 初始化栈
// 参数: s - 指向栈结构的指针, maxSize - 栈的最大容量
// 返回: 成功返回true，失败返回false
bool initStack(Stack *s, int maxSize) {
    // 分配内存给栈的数组
    s->data = (int*)malloc(sizeof(int) * maxSize);
    if (s->data == NULL) {
        // 内存分配失败
        return false;
    }
    s->top = -1;        // 初始化栈顶为-1，表示空栈
    s->maxSize = maxSize; // 设置最大容量
    return true;
}

// 入栈操作
// 参数: s - 指向栈结构的指针, value - 要入栈的值
// 返回: 成功返回true，失败(栈满)返回false
bool push(Stack *s, int value) {
    // 检查栈是否已满
    if (s->top == s->maxSize - 1) {
        printf("栈已满，无法入栈!\n");
        return false;
    }
    s->top++;           // 栈顶指针加1
    s->data[s->top] = value; // 将值存入栈顶位置
    return true;
}

// 出栈操作
// 参数: s - 指向栈结构的指针, value - 用于接收出栈值的指针
// 返回: 成功返回true，失败(栈空)返回false
bool pop(Stack *s, int *value) {
    // 检查栈是否为空
    if (s->top == -1) {
        printf("栈为空，无法出栈!\n");
        return false;
    }
    *value = s->data[s->top]; // 获取栈顶元素
    s->top--;                 // 栈顶指针减1
    return true;
}

// 获取栈顶元素(不弹出)
// 参数: s - 指向栈结构的指针, value - 用于接收栈顶值的指针
// 返回: 成功返回true，失败(栈空)返回false
bool peek(Stack *s, int *value) {
    // 检查栈是否为空
    if (s->top == -1) {
        printf("栈为空，无法获取栈顶元素!\n");
        return false;
    }
    *value = s->data[s->top]; // 获取栈顶元素
    return true;
}

// 判断栈是否为空
// 参数: s - 指向栈结构的指针
// 返回: 空栈返回true，非空返回false
bool isEmpty(Stack *s) {
    return s->top == -1;
}

// 判断栈是否已满
// 参数: s - 指向栈结构的指针
// 返回: 满栈返回true，未满返回false
bool isFull(Stack *s) {
    return s->top == s->maxSize - 1;
}

// 获取栈中元素个数
// 参数: s - 指向栈结构的指针
// 返回: 当前栈中元素个数
int size(Stack *s) {
    return s->top + 1;
}

// 销毁栈，释放内存
// 参数: s - 指向栈结构的指针
void destroyStack(Stack *s) {
    if (s->data != NULL) {
        free(s->data);  // 释放动态分配的内存
        s->data = NULL;
        s->top = -1;
        s->maxSize = 0;
    }
}

// 测试函数
int main() {
    Stack myStack;
    int value;
    
    // 初始化一个最大容量为5的栈
    if (!initStack(&myStack, 5)) {
        printf("栈初始化失败!\n");
        return -1;
    }
    
    // 测试入栈
    printf("入栈操作:\n");
    for (int i = 1; i <= 5; i++) {
        if (push(&myStack, i * 10)) {
            printf("成功入栈: %d\n", i * 10);
        }
    }
    
    // 尝试再次入栈(应该失败)
    push(&myStack, 60);
    
    // 测试获取栈大小
    printf("当前栈大小: %d\n", size(&myStack));
    
    // 测试获取栈顶元素
    if (peek(&myStack, &value)) {
        printf("栈顶元素: %d\n", value);
    }
    
    // 测试出栈
    printf("出栈操作:\n");
    while (!isEmpty(&myStack)) {
        if (pop(&myStack, &value)) {
            printf("成功出栈: %d\n", value);
        }
    }
    
    // 尝试再次出栈(应该失败)
    pop(&myStack, &value);
    
    // 销毁栈
    destroyStack(&myStack);
    
    return 0;
}
```

顺序栈的优点是实现简单，随机访问效率高；缺点是需要预先确定栈的最大容量，可能导致空间浪费或溢出。另外，顺序栈的内存是连续的，可能在某些情况下难以分配大块连续内存。

### 3.3 链栈实现

链栈是使用链表来实现的栈。每个节点包含数据和指向下一个节点的指针。栈顶就是链表的头节点，这样入栈和出栈操作都可以在O(1)时间内完成。

#### 链栈实现方法表

| 功能名称 | 实例调用方法                         | 具体功能与注意事项                       |
| -------- | ------------------------------------ | ---------------------------------------- |
| 初始化栈 | `initLinkedStack(LNode **top)`       | 创建一个空栈，top指向NULL                |
| 入栈操作 | `pushLinked(LNode **top, int value)` | 在栈顶插入新节点，无需检查容量限制       |
| 出栈操作 | `popLinked(LNode **top, int *value)` | 移除栈顶节点并返回其值，需检查栈是否为空 |
| 获取栈顶 | `peekLinked(LNode *top, int *value)` | 获取栈顶元素但不移除，需检查栈是否为空   |
| 判断空栈 | `isEmptyLinked(LNode *top)`          | 检查栈是否为空，当top==NULL时栈为空      |
| 获取大小 | `sizeLinked(LNode *top)`             | 遍历链表计算元素个数，时间复杂度O(n)     |
| 销毁栈   | `destroyLinkedStack(LNode **top)`    | 释放所有节点内存                         |

下面是链栈的完整实现：

```c
#include <stdio.h>
#include <stdlib.h>
#include <stdbool.h>

// 定义链栈节点结构体
typedef struct Node {
    int data;               // 数据域
    struct Node *next;      // 指针域，指向下一个节点
} LNode;

// 初始化链栈
// 参数: top - 指向栈顶指针的指针
// 返回: 无，直接修改top指向NULL
void initLinkedStack(LNode **top) {
    *top = NULL;  // 空栈，栈顶指针为NULL
}

// 入栈操作
// 参数: top - 指向栈顶指针的指针, value - 要入栈的值
// 返回: 成功返回true，失败(内存分配失败)返回false
bool pushLinked(LNode **top, int value) {
    // 创建新节点
    LNode *newNode = (LNode*)malloc(sizeof(LNode));
    if (newNode == NULL) {
        // 内存分配失败
        printf("内存分配失败!\n");
        return false;
    }
    newNode->data = value;   // 设置新节点的数据
    newNode->next = *top;    // 新节点指向原来的栈顶
    *top = newNode;          // 更新栈顶为新节点
    return true;
}

// 出栈操作
// 参数: top - 指向栈顶指针的指针, value - 用于接收出栈值的指针
// 返回: 成功返回true，失败(栈空)返回false
bool popLinked(LNode **top, int *value) {
    // 检查栈是否为空
    if (*top == NULL) {
        printf("栈为空，无法出栈!\n");
        return false;
    }
    LNode *temp = *top;      // 保存当前栈顶节点
    *value = temp->data;     // 获取栈顶元素的值
    *top = (*top)->next;     // 更新栈顶为下一个节点
    free(temp);              // 释放原栈顶节点内存
    return true;
}

// 获取栈顶元素(不弹出)
// 参数: top - 栈顶指针, value - 用于接收栈顶值的指针
// 返回: 成功返回true，失败(栈空)返回false
bool peekLinked(LNode *top, int *value) {
    // 检查栈是否为空
    if (top == NULL) {
        printf("栈为空，无法获取栈顶元素!\n");
        return false;
    }
    *value = top->data;      // 获取栈顶元素的值
    return true;
}

// 判断栈是否为空
// 参数: top - 栈顶指针
// 返回: 空栈返回true，非空返回false
bool isEmptyLinked(LNode *top) {
    return top == NULL;
}

// 获取栈中元素个数
// 参数: top - 栈顶指针
// 返回: 当前栈中元素个数
int sizeLinked(LNode *top) {
    int count = 0;
    LNode *current = top;
    // 遍历链表计数
    while (current != NULL) {
        count++;
        current = current->next;
    }
    return count;
}

// 销毁链栈，释放所有节点内存
// 参数: top - 指向栈顶指针的指针
void destroyLinkedStack(LNode **top) {
    LNode *current = *top;
    LNode *next;
    // 逐个释放节点
    while (current != NULL) {
        next = current->next;
        free(current);
        current = next;
    }
    *top = NULL;  // 栈顶置为NULL
}

// 打印栈中所有元素(从栈顶到栈底)
// 参数: top - 栈顶指针
void printStack(LNode *top) {
    if (top == NULL) {
        printf("栈为空\n");
        return;
    }
    printf("栈中元素(从栈顶到栈底): ");
    LNode *current = top;
    while (current != NULL) {
        printf("%d ", current->data);
        current = current->next;
    }
    printf("\n");
}

// 测试函数
int main() {
    LNode *myStack;
    int value;
    
    // 初始化链栈
    initLinkedStack(&myStack);
    
    // 测试入栈
    printf("入栈操作:\n");
    for (int i = 1; i <= 5; i++) {
        if (pushLinked(&myStack, i * 10)) {
            printf("成功入栈: %d\n", i * 10);
        }
    }
    
    // 打印栈内容
    printStack(myStack);
    
    // 测试获取栈大小
    printf("当前栈大小: %d\n", sizeLinked(myStack));
    
    // 测试获取栈顶元素
    if (peekLinked(myStack, &value)) {
        printf("栈顶元素: %d\n", value);
    }
    
    // 测试出栈
    printf("出栈操作:\n");
    while (!isEmptyLinked(myStack)) {
        if (popLinked(&myStack, &value)) {
            printf("成功出栈: %d\n", value);
        }
    }
    
    // 尝试再次出栈(应该失败)
    popLinked(&myStack, &value);
    
    // 再次入栈测试
    printf("\n再次入栈测试:\n");
    pushLinked(&myStack, 100);
    pushLinked(&myStack, 200);
    printStack(myStack);
    
    // 销毁栈
    destroyLinkedStack(&myStack);
    
    return 0;
}
```

链栈的优点是不需要预先确定栈的大小，可以动态增长，内存利用率高；缺点是每个节点需要额外的指针空间，且不能随机访问元素。链栈适合栈大小变化较大或不确定的场景。

### 3.4 栈的应用实例

栈在计算机科学中有许多重要应用，下面介绍几个经典的应用实例。

#### 括号匹配检查

检查表达式中的括号是否正确匹配是栈的一个典型应用。算法思路是：遍历表达式，遇到左括号就入栈，遇到右括号就检查栈顶是否为对应的左括号，如果是则出栈，否则不匹配。

```c
#include <stdio.h>
#include <stdlib.h>
#include <stdbool.h>
#include <string.h>

// 定义栈结构体(简化版，仅用于字符)
typedef struct {
    char *data;
    int top;
    int maxSize;
} CharStack;

// 初始化栈
bool initCharStack(CharStack *s, int maxSize) {
    s->data = (char*)malloc(sizeof(char) * maxSize);
    if (s->data == NULL) return false;
    s->top = -1;
    s->maxSize = maxSize;
    return true;
}

// 入栈
bool pushChar(CharStack *s, char value) {
    if (s->top == s->maxSize - 1) return false;
    s->data[++s->top] = value;
    return true;
}

// 出栈
bool popChar(CharStack *s, char *value) {
    if (s->top == -1) return false;
    *value = s->data[s->top--];
    return true;
}

// 判断栈是否为空
bool isEmptyChar(CharStack *s) {
    return s->top == -1;
}

// 销毁栈
void destroyCharStack(CharStack *s) {
    if (s->data != NULL) {
        free(s->data);
        s->data = NULL;
    }
}

// 检查两个括号是否匹配
bool isMatchingPair(char open, char close) {
    return (open == '(' && close == ')') ||
           (open == '[' && close == ']') ||
           (open == '{' && close == '}');
}

// 检查表达式括号是否匹配
bool checkBrackets(const char *expression) {
    int len = strlen(expression);
    CharStack stack;
    
    // 初始化栈，大小为表达式长度
    if (!initCharStack(&stack, len)) {
        printf("栈初始化失败!\n");
        return false;
    }
    
    for (int i = 0; i < len; i++) {
        char ch = expression[i];
        
        // 如果是左括号，入栈
        if (ch == '(' || ch == '[' || ch == '{') {
            pushChar(&stack, ch);
        }
        // 如果是右括号
        else if (ch == ')' || ch == ']' || ch == '}') {
            char topChar;
            // 栈为空，没有匹配的左括号
            if (isEmptyChar(&stack)) {
                destroyCharStack(&stack);
                return false;
            }
            // 弹出栈顶元素
            popChar(&stack, &topChar);
            // 检查是否匹配
            if (!isMatchingPair(topChar, ch)) {
                destroyCharStack(&stack);
                return false;
            }
        }
        // 其他字符忽略
    }
    
    // 最后栈应该为空
    bool result = isEmptyChar(&stack);
    destroyCharStack(&stack);
    return result;
}

// 测试函数
int main() {
    char expressions[][50] = {
        "((()))",
        "([{}])",
        "((())",
        "([)]",
        "{}",
        ""
    };
    
    int numTests = sizeof(expressions) / sizeof(expressions[0]);
    
    for (int i = 0; i < numTests; i++) {
        printf("表达式 \"%s\": ", expressions[i]);
        if (checkBrackets(expressions[i])) {
            printf("括号匹配正确\n");
        } else {
            printf("括号匹配错误\n");
        }
    }
    
    return 0;
}
```

#### 表达式求值（后缀表达式）

后缀表达式（逆波兰表达式）是一种不需要括号的表达式表示方法，可以用栈高效求值。例如，中缀表达式 "3 + 4 * 2" 的后缀表达式是 "3 4 2 * +"。

```c
#include <stdio.h>
#include <stdlib.h>
#include <stdbool.h>
#include <string.h>
#include <ctype.h>

// 使用之前定义的整数栈
typedef struct {
    int *data;
    int top;
    int maxSize;
} IntStack;

bool initIntStack(IntStack *s, int maxSize) {
    s->data = (int*)malloc(sizeof(int) * maxSize);
    if (s->data == NULL) return false;
    s->top = -1;
    s->maxSize = maxSize;
    return true;
}

bool pushInt(IntStack *s, int value) {
    if (s->top == s->maxSize - 1) return false;
    s->data[++s->top] = value;
    return true;
}

bool popInt(IntStack *s, int *value) {
    if (s->top == -1) return false;
    *value = s->data[s->top--];
    return true;
}

bool isEmptyInt(IntStack *s) {
    return s->top == -1;
}

void destroyIntStack(IntStack *s) {
    if (s->data != NULL) {
        free(s->data);
        s->data = NULL;
    }
}

// 判断字符是否为运算符
bool isOperator(char c) {
    return c == '+' || c == '-' || c == '*' || c == '/';
}

// 执行运算
int performOperation(int operand1, int operand2, char operator) {
    switch (operator) {
        case '+': return operand1 + operand2;
        case '-': return operand1 - operand2;
        case '*': return operand1 * operand2;
        case '/': 
            if (operand2 == 0) {
                printf("除零错误!\n");
                return 0;
            }
            return operand1 / operand2;
        default: return 0;
    }
}

// 计算后缀表达式的值
// 表达式格式: 数字和运算符用空格分隔，如 "3 4 2 * +"
int evaluatePostfix(const char *postfix) {
    IntStack stack;
    if (!initIntStack(&stack, 100)) { // 假设最大100个元素
        printf("栈初始化失败!\n");
        return 0;
    }
    
    char *expr = strdup(postfix); // 复制字符串以便使用strtok
    char *token = strtok(expr, " ");
    
    while (token != NULL) {
        // 如果是数字
        if (isdigit(token[0]) || (token[0] == '-' && strlen(token) > 1)) {
            int num = atoi(token);
            pushInt(&stack, num);
        }
        // 如果是运算符
        else if (isOperator(token[0])) {
            int operand2, operand1, result;
            // 需要两个操作数
            if (!popInt(&stack, &operand2) || !popInt(&stack, &operand1)) {
                printf("表达式格式错误!\n");
                destroyIntStack(&stack);
                free(expr);
                return 0;
            }
            result = performOperation(operand1, operand2, token[0]);
            pushInt(&stack, result);
        }
        token = strtok(NULL, " ");
    }
    
    // 最终结果应该在栈顶，且栈中只有一个元素
    int result;
    if (popInt(&stack, &result) && isEmptyInt(&stack)) {
        destroyIntStack(&stack);
        free(expr);
        return result;
    } else {
        printf("表达式格式错误!\n");
        destroyIntStack(&stack);
        free(expr);
        return 0;
    }
}

// 测试函数
int main() {
    char expressions[][50] = {
        "3 4 +",           // 7
        "3 4 2 * +",       // 11
        "15 7 1 1 + - / 3 * 2 1 1 + + -", // 5
        "5 1 2 + 4 * + 3 -" // 14
    };
    
    int numTests = sizeof(expressions) / sizeof(expressions[0]);
    
    for (int i = 0; i < numTests; i++) {
        printf("后缀表达式 \"%s\" = %d\n", 
               expressions[i], evaluatePostfix(expressions[i]));
    }
    
    return 0;
}
```

#### 函数调用模拟

栈还可以用来模拟函数调用过程。每次函数调用时，将函数信息压入调用栈；函数返回时，从调用栈弹出。

```c
#include <stdio.h>
#include <stdlib.h>
#include <stdbool.h>
#include <string.h>

// 函数调用帧结构
typedef struct {
    char functionName[50];  // 函数名
    int returnAddress;      // 返回地址(简化为整数)
    int parameters[10];     // 参数(最多10个)
    int paramCount;         // 参数个数
} CallFrame;

// 调用栈结构
typedef struct {
    CallFrame *frames;
    int top;
    int maxSize;
} CallStack;

// 初始化调用栈
bool initCallStack(CallStack *cs, int maxSize) {
    cs->frames = (CallFrame*)malloc(sizeof(CallFrame) * maxSize);
    if (cs->frames == NULL) return false;
    cs->top = -1;
    cs->maxSize = maxSize;
    return true;
}

// 函数调用(入栈)
bool callFunction(CallStack *cs, const char *funcName, 
                  int returnAddr, int *params, int paramCount) {
    if (cs->top == cs->maxSize - 1) {
        printf("调用栈溢出!\n");
        return false;
    }
    
    cs->top++;
    strcpy(cs->frames[cs->top].functionName, funcName);
    cs->frames[cs->top].returnAddress = returnAddr;
    
    // 复制参数
    if (paramCount > 10) paramCount = 10; // 限制参数数量
    cs->frames[cs->top].paramCount = paramCount;
    for (int i = 0; i < paramCount; i++) {
        cs->frames[cs->top].parameters[i] = params[i];
    }
    
    printf("调用函数: %s (返回地址: %d)\n", funcName, returnAddr);
    return true;
}

// 函数返回(出栈)
bool returnFromFunction(CallStack *cs) {
    if (cs->top == -1) {
        printf("调用栈为空!\n");
        return false;
    }
    
    printf("从函数 %s 返回 (返回地址: %d)\n", 
           cs->frames[cs->top].functionName, 
           cs->frames[cs->top].returnAddress);
    cs->top--;
    return true;
}

// 打印当前调用栈
void printCallStack(CallStack *cs) {
    if (cs->top == -1) {
        printf("调用栈为空\n");
        return;
    }
    
    printf("当前调用栈 (从栈顶到栈底):\n");
    for (int i = cs->top; i >= 0; i--) {
        printf("  %s (返回地址: %d)\n", 
               cs->frames[i].functionName, 
               cs->frames[i].returnAddress);
    }
}

// 销毁调用栈
void destroyCallStack(CallStack *cs) {
    if (cs->frames != NULL) {
        free(cs->frames);
        cs->frames = NULL;
    }
}

// 测试函数
int main() {
    CallStack callStack;
    if (!initCallStack(&callStack, 10)) {
        printf("调用栈初始化失败!\n");
        return -1;
    }
    
    // 模拟函数调用序列
    int params1[] = {10, 20};
    callFunction(&callStack, "main", 0, NULL, 0);
    callFunction(&callStack, "functionA", 100, params1, 2);
    
    int params2[] = {30};
    callFunction(&callStack, "functionB", 200, params2, 1);
    
    // 打印当前调用栈
    printCallStack(&callStack);
    
    // 模拟函数返回
    returnFromFunction(&callStack);
    returnFromFunction(&callStack);
    returnFromFunction(&callStack);
    
    // 尝试再次返回(应该失败)
    returnFromFunction(&callStack);
    
    destroyCallStack(&callStack);
    return 0;
}
```

栈的应用非常广泛，除了上述例子，还包括：

- 浏览器的前进/后退功能
- 撤销(Undo)操作
- 深度优先搜索(DFS)
- 语法解析
- 内存管理

掌握栈的概念和实现对于理解计算机系统的工作原理非常重要。无论是顺序栈还是链栈，都有各自的适用场景，选择哪种实现方式取决于具体需求。
