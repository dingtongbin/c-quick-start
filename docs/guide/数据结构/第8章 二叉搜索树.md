### 8.1 二叉搜索树概念

二叉搜索树（Binary Search Tree，简称 BST）是一种特殊的二叉树结构，它在查找、插入和删除操作上具有良好的平均时间复杂度。如果你已经掌握了普通二叉树的基本知识，那么 BST 就像是给二叉树加了一套“交通规则”——左小右大，井然有序。

| 技术点       | 说明                                                         |
| ------------ | ------------------------------------------------------------ |
| 当前数据结构 | 二叉搜索树（BST）                                            |
| 依赖基础     | C语言指针、结构体、递归                                      |
| 适用场景     | 动态查找表、需要频繁查找/插入/删除的有序数据集合             |
| 核心性质     | 对任意节点，其左子树所有节点值 < 该节点值 < 右子树所有节点值 |

#### 二叉搜索树的核心特性

- **有序性**：中序遍历结果为严格升序序列（假设无重复值）
- **唯一性**：通常不允许重复键值（也可通过计数器支持重复）
- **高效性**：理想情况下（树平衡），查找/插入/删除时间复杂度为 O(log n)

下面用表格总结几个关键操作的概念：

| 功能名称 | 操作描述                                     | 注意事项                                   |
| -------- | -------------------------------------------- | ------------------------------------------ |
| 查找     | 从根开始，比当前节点小则走左，大则走右       | 若到达空指针仍未找到，说明元素不存在       |
| 插入     | 按查找路径走到空位置，新建节点插入           | 必须保持 BST 性质，不能破坏结构            |
| 删除     | 分三种情况：无子节点、一个子节点、两个子节点 | 删除有两个子节点时，需用中序后继或前驱替代 |

我们先来看一个最简单的 BST 节点定义和创建示例。

```c
#include <stdio.h>
#include <stdlib.h>

// 定义二叉搜索树节点结构体
struct TreeNode {
    int data;                    // 节点存储的数据
    struct TreeNode* left;       // 左子节点指针
    struct TreeNode* right;      // 右子节点指针
};

// 创建新节点的辅助函数
struct TreeNode* createNode(int value) {
    // 动态分配内存创建新节点
    struct TreeNode* newNode = (struct TreeNode*)malloc(sizeof(struct TreeNode));
    if (newNode == NULL) {
        fprintf(stderr, "内存分配失败！\n");  // 错误处理：内存不足
        return NULL;
    }
    newNode->data = value;       // 初始化数据域
    newNode->left = NULL;        // 左子树初始为空
    newNode->right = NULL;       // 右子树初始为空
    return newNode;              // 返回新节点指针
}

// 示例：创建一个简单的 BST（手动构建）
int main() {
    // 创建根节点，值为 50
    struct TreeNode* root = createNode(50);
    if (root == NULL) return -1; // 安全检查

    // 插入左子节点（30 < 50）
    root->left = createNode(30);
    // 插入右子节点（70 > 50）
    root->right = createNode(70);

    // 打印验证
    printf("根节点: %d\n", root->data);
    printf("左子节点: %d\n", root->left->data);
    printf("右子节点: %d\n", root->right->data);

    // 释放内存（简单示例，实际应递归释放整棵树）
    free(root->left);
    free(root->right);
    free(root);

    return 0;
}
```

这段代码展示了如何定义 BST 节点并手动构建一个包含三个节点的小树。虽然还没实现自动插入逻辑，但它清晰地体现了 BST 的结构特征：左小右大。

**注意事项**：

- 所有动态分配的内存必须手动释放，否则会造成内存泄漏
- `createNode` 函数必须检查 `malloc` 是否成功，这是 C 语言健壮编程的基本要求
- 此处的树是硬编码构建的，后续章节将实现自动按规则插入的函数

这节讲了二叉搜索树的基本概念和节点结构定义，它是后续查找、插入、删除操作的基础。理解“左小右大”的有序性是掌握 BST 的关键。
