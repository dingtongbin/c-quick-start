# 第16章 简单排序

## 16.1 冒泡排序

### 版本信息表格

| 项目       | 信息                                         |
| ---------- | -------------------------------------------- |
| C语言标准  | C99/C11                                      |
| 编译器要求 | GCC 4.0+ 或 Clang 3.0+                       |
| 适用平台   | 跨平台                                       |
| 使用场景   | 小规模数据排序、教学演示、稳定性要求高的场景 |

冒泡排序是最基础的排序算法之一，虽然效率不高，但理解起来非常直观。它的核心思想就是"大泡泡往上冒"——通过相邻元素的比较和交换，让较大的元素逐步"冒泡"到数组的末尾。

### 实例方法表格

| 功能名称     | 实例调用方法                    | 具体功能与注意事项                                  |
| ------------ | ------------------------------- | --------------------------------------------------- |
| 基础冒泡排序 | `bubble_sort(arr, n)`           | 对数组进行升序排序，时间复杂度O(n²)，空间复杂度O(1) |
| 优化冒泡排序 | `optimized_bubble_sort(arr, n)` | 添加提前终止条件，最好情况时间复杂度O(n)            |
| 降序冒泡排序 | `bubble_sort_desc(arr, n)`      | 实现降序排列，只需修改比较条件                      |

### 使用示例

```c
#include <stdio.h>
#include <stdbool.h>

// 冒泡排序函数 - 升序排列
// arr: 待排序的整型数组
// n: 数组元素个数
void bubble_sort(int arr[], int n) {
    // 外层循环控制排序轮数，n个元素需要n-1轮
    for (int i = 0; i < n - 1; i++) {
        // 内层循环进行相邻元素比较和交换
        for (int j = 0; j < n - 1 - i; j++) {
            // 如果前一个元素大于后一个元素，则交换
            if (arr[j] > arr[j + 1]) {
                int temp = arr[j];      // 临时变量存储arr[j]
                arr[j] = arr[j + 1];    // 将较小的元素放到前面
                arr[j + 1] = temp;      // 将较大的元素放到后面
            }
        }
    }
}

// 优化版冒泡排序 - 添加提前终止条件
// arr: 待排序的整型数组
// n: 数组元素个数
void optimized_bubble_sort(int arr[], int n) {
    // 外层循环控制排序轮数
    for (int i = 0; i < n - 1; i++) {
        bool swapped = false;  // 标记本轮是否发生交换
        
        // 内层循环进行相邻元素比较
        for (int j = 0; j < n - 1 - i; j++) {
            if (arr[j] > arr[j + 1]) {
                // 执行交换操作
                int temp = arr[j];
                arr[j] = arr[j + 1];
                arr[j + 1] = temp;
                swapped = true;  // 标记发生了交换
            }
        }
        
        // 如果本轮没有发生交换，说明数组已经有序
        if (!swapped) {
            break;  // 提前终止排序
        }
    }
}

// 打印数组函数
// arr: 要打印的数组
// n: 数组元素个数
void print_array(int arr[], int n) {
    for (int i = 0; i < n; i++) {
        printf("%d ", arr[i]);  // 逐个打印数组元素
    }
    printf("\n");  // 换行
}

// 主函数 - 演示冒泡排序
int main() {
    // 测试数据
    int arr[] = {64, 34, 25, 12, 22, 11, 90};
    int n = sizeof(arr) / sizeof(arr[0]);  // 计算数组长度
    
    printf("原始数组: ");
    print_array(arr, n);  // 打印原始数组
    
    // 调用优化版冒泡排序
    optimized_bubble_sort(arr, n);
    
    printf("排序后数组: ");
    print_array(arr, n);  // 打印排序后的数组
    
    return 0;  // 程序正常结束
}
```

### 注意事项

- **输入验证**：在实际应用中应该检查数组是否为空（n <= 0）的情况
- **稳定性**：冒泡排序是稳定的排序算法，相等元素的相对位置不会改变
- **性能特点**：虽然时间复杂度较高，但在数据基本有序的情况下表现较好
- **内存使用**：原地排序，只需要常数级别的额外空间

冒泡排序通过重复遍历数组，比较相邻元素并交换位置来实现排序。虽然效率不高，但其简单性和稳定性使其成为学习排序算法的良好起点。

## 16.2 选择排序

### 版本信息表格

| 项目       | 信息                                   |
| ---------- | -------------------------------------- |
| C语言标准  | C99/C11                                |
| 编译器要求 | GCC 4.0+ 或 Clang 3.0+                 |
| 适用平台   | 跨平台                                 |
| 使用场景   | 小规模数据排序、内存受限环境、教学演示 |

选择排序的核心思想很直接：每一轮都从待排序的部分中找到最小（或最大）的元素，然后把它放到已排序部分的末尾。就像你在一堆衣服里一件一件挑选最喜欢的那样，每次选出当前最好的。

### 实例方法表格

| 功能名称       | 实例调用方法                      | 具体功能与注意事项                              |
| -------------- | --------------------------------- | ----------------------------------------------- |
| 基础选择排序   | `selection_sort(arr, n)`          | 升序排列，时间复杂度始终为O(n²)，空间复杂度O(1) |
| 选择最大值排序 | `selection_sort_max(arr, n)`      | 每次选择最大值放到末尾                          |
| 双向选择排序   | `bidirectional_selection(arr, n)` | 同时找最小值和最大值，减少比较次数              |

### 使用示例

```c
#include <stdio.h>
#include <limits.h>

// 选择排序函数 - 升序排列
// arr: 待排序的整型数组
// n: 数组元素个数
void selection_sort(int arr[], int n) {
    // 外层循环控制已排序部分的边界
    // i表示当前要填充的位置
    for (int i = 0; i < n - 1; i++) {
        int min_index = i;  // 假设当前位置就是最小值的位置
        
        // 在未排序部分寻找真正的最小值
        for (int j = i + 1; j < n; j++) {
            // 如果找到更小的元素，更新最小值索引
            if (arr[j] < arr[min_index]) {
                min_index = j;  // 记录新的最小值位置
            }
        }
        
        // 如果最小值不在当前位置，执行交换
        if (min_index != i) {
            int temp = arr[i];          // 保存当前位置的值
            arr[i] = arr[min_index];    // 将最小值放到当前位置
            arr[min_index] = temp;      // 将原位置的值放到最小值位置
        }
    }
}

// 优化版选择排序 - 同时找最小值和最大值
// arr: 待排序的整型数组
// n: 数组元素个数
void bidirectional_selection(int arr[], int n) {
    int left = 0;           // 左边界
    int right = n - 1;      // 右边界
    
    // 当左边界小于右边界时继续排序
    while (left < right) {
        int min_index = left;   // 最小值索引
        int max_index = left;   // 最大值索引
        
        // 在当前范围内同时找最小值和最大值
        for (int i = left; i <= right; i++) {
            if (arr[i] < arr[min_index]) {
                min_index = i;  // 更新最小值位置
            }
            if (arr[i] > arr[max_index]) {
                max_index = i;  // 更新最大值位置
            }
        }
        
        // 将最小值交换到左边界
        if (min_index != left) {
            int temp = arr[left];
            arr[left] = arr[min_index];
            arr[min_index] = temp;
            
            // 如果最大值原来在left位置，现在被交换到了min_index
            if (max_index == left) {
                max_index = min_index;
            }
        }
        
        // 将最大值交换到右边界
        if (max_index != right) {
            int temp = arr[right];
            arr[right] = arr[max_index];
            arr[max_index] = temp;
        }
        
        // 缩小排序范围
        left++;
        right--;
    }
}

// 打印数组函数
// arr: 要打印的数组
// n: 数组元素个数
void print_array(int arr[], int n) {
    for (int i = 0; i < n; i++) {
        printf("%d ", arr[i]);  // 逐个打印数组元素
    }
    printf("\n");  // 换行
}

// 主函数 - 演示选择排序
int main() {
    // 测试数据
    int arr[] = {64, 34, 25, 12, 22, 11, 90};
    int n = sizeof(arr) / sizeof(arr[0]);  // 计算数组长度
    
    printf("原始数组: ");
    print_array(arr, n);  // 打印原始数组
    
    // 调用基础选择排序
    selection_sort(arr, n);
    
    printf("排序后数组: ");
    print_array(arr, n);  // 打印排序后的数组
    
    // 测试双向选择排序
    int arr2[] = {64, 34, 25, 12, 22, 11, 90};
    printf("\n原始数组(双向): ");
    print_array(arr2, 7);
    
    bidirectional_selection(arr2, 7);
    printf("排序后数组(双向): ");
    print_array(arr2, 7);
    
    return 0;  // 程序正常结束
}
```

### 注意事项

- **交换次数**：选择排序的交换次数最多为n-1次，比冒泡排序少很多
- **不稳定性**：选择排序是不稳定的排序算法，相等元素的相对位置可能改变
- **性能一致性**：无论输入数据如何，时间复杂度都是O(n²)
- **适用场景**：当写操作（交换）代价很高时，选择排序可能比冒泡排序更合适

选择排序通过每次选择未排序部分的最小元素并放置到正确位置来实现排序。虽然时间复杂度较高，但其交换次数少的特点在某些特定场景下很有价值。

## 16.3 插入排序

### 版本信息表格

| 项目       | 信息                                           |
| ---------- | ---------------------------------------------- |
| C语言标准  | C99/C11                                        |
| 编译器要求 | GCC 4.0+ 或 Clang 3.0+                         |
| 适用平台   | 跨平台                                         |
| 使用场景   | 小规模数据、基本有序的数据、在线算法、教学演示 |

插入排序的工作方式就像我们整理扑克牌一样：每次从牌堆中拿出一张牌，然后在手中已有的牌中找到合适的位置插入。这种方法对于小规模数据或者基本有序的数据特别有效。

### 实例方法表格

| 功能名称     | 实例调用方法                    | 具体功能与注意事项                    |
| ------------ | ------------------------------- | ------------------------------------- |
| 基础插入排序 | `insertion_sort(arr, n)`        | 升序排列，最好情况O(n)，最坏情况O(n²) |
| 二分插入排序 | `binary_insertion_sort(arr, n)` | 使用二分查找优化查找插入位置          |
| 降序插入排序 | `insertion_sort_desc(arr, n)`   | 实现降序排列                          |

### 使用示例

```c
#include <stdio.h>

// 插入排序函数 - 升序排列
// arr: 待排序的整型数组
// n: 数组元素个数
void insertion_sort(int arr[], int n) {
    // 从第二个元素开始（索引1），因为第一个元素可以认为已经有序
    for (int i = 1; i < n; i++) {
        int key = arr[i];  // 当前要插入的元素
        int j = i - 1;     // 已排序部分的最后一个元素索引
        
        // 在已排序部分中找到key的正确位置
        // 从右向左移动比key大的元素
        while (j >= 0 && arr[j] > key) {
            arr[j + 1] = arr[j];  // 将较大的元素向右移动
            j--;                  // 继续向前比较
        }
        
        // 将key插入到正确位置
        arr[j + 1] = key;
    }
}

// 二分插入排序 - 优化查找插入位置
// arr: 待排序的整型数组
// n: 数组元素个数
void binary_insertion_sort(int arr[], int n) {
    // 从第二个元素开始处理
    for (int i = 1; i < n; i++) {
        int key = arr[i];  // 当前要插入的元素
        
        // 使用二分查找找到插入位置
        int left = 0;
        int right = i - 1;
        
        // 二分查找过程
        while (left <= right) {
            int mid = (left + right) / 2;
            if (arr[mid] > key) {
                right = mid - 1;  // 在左半部分查找
            } else {
                left = mid + 1;   // 在右半部分查找
            }
        }
        
        // 将元素从插入位置到i-1的位置都向右移动一位
        for (int j = i - 1; j >= left; j--) {
            arr[j + 1] = arr[j];
        }
        
        // 插入元素到正确位置
        arr[left] = key;
    }
}

// 打印数组函数
// arr: 要打印的数组
// n: 数组元素个数
void print_array(int arr[], int n) {
    for (int i = 0; i < n; i++) {
        printf("%d ", arr[i]);  // 逐个打印数组元素
    }
    printf("\n");  // 换行
}

// 主函数 - 演示插入排序
int main() {
    // 测试数据
    int arr[] = {64, 34, 25, 12, 22, 11, 90};
    int n = sizeof(arr) / sizeof(arr[0]);  // 计算数组长度
    
    printf("原始数组: ");
    print_array(arr, n);  // 打印原始数组
    
    // 调用基础插入排序
    insertion_sort(arr, n);
    
    printf("排序后数组: ");
    print_array(arr, n);  // 打印排序后的数组
    
    // 测试二分插入排序
    int arr2[] = {64, 34, 25, 12, 22, 11, 90};
    printf("\n原始数组(二分): ");
    print_array(arr2, 7);
    
    binary_insertion_sort(arr2, 7);
    printf("排序后数组(二分): ");
    print_array(arr2, 7);
    
    return 0;  // 程序正常结束
}
```

### 注意事项

- **在线算法**：插入排序是在线算法，可以在接收数据的同时进行排序
- **稳定性**：插入排序是稳定的排序算法
- **自适应性**：对于基本有序的数据，插入排序效率很高
- **空间复杂度**：原地排序，空间复杂度为O(1)
- **实际应用**：许多高效的排序算法（如快速排序）在小规模子数组上会切换到插入排序

插入排序通过将每个元素插入到已排序部分的正确位置来实现排序。它对小规模数据和基本有序数据表现优异，是许多高级排序算法的重要组成部分。

## 16.4 希尔排序

### 版本信息表格

| 项目       | 信息                                       |
| ---------- | ------------------------------------------ |
| C语言标准  | C99/C11                                    |
| 编译器要求 | GCC 4.0+ 或 Clang 3.0+                     |
| 适用平台   | 跨平台                                     |
| 使用场景   | 中等规模数据排序、需要比简单排序更快的场景 |

希尔排序是插入排序的一个聪明的改进版本，也被称为"缩小增量排序"。它的核心思想是先让数组中的元素变得"大致有序"，然后再进行精细排序。这就像先用粗砂纸打磨木头，再用细砂纸抛光一样。

### 实例方法表格

| 功能名称            | 实例调用方法                                 | 具体功能与注意事项                      |
| ------------------- | -------------------------------------------- | --------------------------------------- |
| 希尔排序(Shell序列) | `shell_sort_shell(arr, n)`                   | 使用Shell原始增量序列：n/2, n/4, ..., 1 |
| 希尔排序(Knuth序列) | `shell_sort_knuth(arr, n)`                   | 使用Knuth序列：(3^k-1)/2，性能更好      |
| 自定义增量希尔排序  | `shell_sort_custom(arr, n, gaps, gap_count)` | 支持自定义增量序列                      |

### 使用示例

```c
#include <stdio.h>
#include <math.h>

// 希尔排序 - 使用Shell原始增量序列
// arr: 待排序的整型数组
// n: 数组元素个数
void shell_sort_shell(int arr[], int n) {
    // 初始化增量为数组长度的一半
    for (int gap = n / 2; gap > 0; gap /= 2) {
        // 对每个增量进行插入排序
        for (int i = gap; i < n; i++) {
            int temp = arr[i];  // 当前要插入的元素
            int j;
            
            // 在间隔为gap的子序列中进行插入排序
            for (j = i; j >= gap && arr[j - gap] > temp; j -= gap) {
                arr[j] = arr[j - gap];  // 向后移动元素
            }
            
            // 插入元素到正确位置
            arr[j] = temp;
        }
    }
}

// 希尔排序 - 使用Knuth增量序列
// arr: 待排序的整型数组
// n: 数组元素个数
void shell_sort_knuth(int arr[], int n) {
    // 计算Knuth序列的最大增量
    int gap = 1;
    while (gap < n / 3) {
        gap = gap * 3 + 1;  // Knuth序列: 1, 4, 13, 40, 121, ...
    }
    
    // 从最大增量开始，逐步减小
    while (gap >= 1) {
        // 对当前增量进行插入排序
        for (int i = gap; i < n; i++) {
            int temp = arr[i];
            int j;
            
            // 在间隔为gap的子序列中进行插入排序
            for (j = i; j >= gap && arr[j - gap] > temp; j -= gap) {
                arr[j] = arr[j - gap];
            }
            
            arr[j] = temp;
        }
        
        gap = gap / 3;  // 下一个Knuth增量
    }
}

// 计算Knuth序列的辅助函数
// n: 数组长度
// 返回Knuth序列数组和序列长度
int* generate_knuth_sequence(int n, int* sequence_length) {
    int* sequence = malloc(sizeof(int) * 20);  // 预分配足够空间
    int gap = 1;
    int index = 0;
    
    // 生成所有小于n的Knuth增量
    while (gap < n) {
        sequence[index++] = gap;
        gap = gap * 3 + 1;
    }
    
    // 反转序列，从大到小
    for (int i = 0; i < index / 2; i++) {
        int temp = sequence[i];
        sequence[i] = sequence[index - 1 - i];
        sequence[index - 1 - i] = temp;
    }
    
    *sequence_length = index;
    return sequence;
}

// 打印数组函数
// arr: 要打印的数组
// n: 数组元素个数
void print_array(int arr[], int n) {
    for (int i = 0; i < n; i++) {
        printf("%d ", arr[i]);  // 逐个打印数组元素
    }
    printf("\n");  // 换行
}

// 主函数 - 演示希尔排序
int main() {
    // 测试数据
    int arr1[] = {64, 34, 25, 12, 22, 11, 90, 88, 76, 50, 42};
    int n1 = sizeof(arr1) / sizeof(arr1[0]);
    
    printf("原始数组: ");
    print_array(arr1, n1);
    
    // 调用Shell序列希尔排序
    shell_sort_shell(arr1, n1);
    
    printf("Shell序列排序后: ");
    print_array(arr1, n1);
    
    // 测试Knuth序列
    int arr2[] = {64, 34, 25, 12, 22, 11, 90, 88, 76, 50, 42};
    int n2 = sizeof(arr2) / sizeof(arr2[0]);
    
    printf("\n原始数组: ");
    print_array(arr2, n2);
    
    shell_sort_knuth(arr2, n2);
    
    printf("Knuth序列排序后: ");
    print_array(arr2, n2);
    
    return 0;
}
```

### 注意事项

- **增量序列选择**：不同的增量序列对性能影响很大，Knuth序列通常比Shell原始序列表现更好
- **稳定性**：希尔排序是不稳定的排序算法
- **时间复杂度**：取决于增量序列的选择，Shell序列为O(n²)，Knuth序列约为O(n^1.25)
- **空间复杂度**：原地排序，空间复杂度为O(1)
- **实际性能**：对于中等规模的数据（几千到几万个元素），希尔排序通常比简单排序快很多

希尔排序通过使用递减的增量序列对数组进行多次插入排序，使得数组逐渐变得有序。这种"先粗后细"的策略大大提高了排序效率，是简单排序算法中的佼佼者。
