# 第17章 高效排序

## 17.1 快速排序

快速排序是一种分治算法，通过选择一个"基准"元素，将数组分为两部分：小于基准的元素和大于基准的元素，然后递归地对这两部分进行排序。它的平均时间复杂度为O(n log n)，是实际应用中最常用的排序算法之一。

### 快速排序核心方法

| 功能名称       | 方法调用                              | 具体功能与注意事项                                           |
| -------------- | ------------------------------------- | ------------------------------------------------------------ |
| 分区操作       | `partition(arr, low, high)`           | 选择基准元素，重新排列数组使得小于基准的在左边，大于基准的在右边。返回基准元素的最终位置。必需参数：数组、起始索引、结束索引 |
| 快速排序主函数 | `quickSort(arr, low, high)`           | 递归实现快速排序。必需参数：数组、起始索引、结束索引。注意递归终止条件 |
| 随机化快速排序 | `randomizedQuickSort(arr, low, high)` | 通过随机选择基准元素来避免最坏情况。可选优化，提高算法稳定性 |

### 使用示例

```c
#include <stdio.h>
#include <stdlib.h>
#include <time.h>

// 分区函数：将数组分为两部分，返回基准元素的位置
// 输入：arr-待排序数组，low-起始索引，high-结束索引
// 输出：基准元素的最终位置索引
int partition(int arr[], int low, int high) {
    // 选择最后一个元素作为基准
    int pivot = arr[high];
    // i指向小于基准元素区域的边界
    int i = low - 1;
    
    // 遍历从low到high-1的所有元素
    for (int j = low; j < high; j++) {
        // 如果当前元素小于或等于基准
        if (arr[j] <= pivot) {
            i++; // 扩展小于基准的区域
            // 交换arr[i]和arr[j]
            int temp = arr[i];
            arr[i] = arr[j];
            arr[j] = temp;
        }
    }
    // 将基准元素放到正确位置
    int temp = arr[i + 1];
    arr[i + 1] = arr[high];
    arr[high] = temp;
    // 返回基准元素的位置
    return i + 1;
}

// 快速排序主函数
// 输入：arr-待排序数组，low-起始索引，high-结束索引
// 输出：无（原地排序）
void quickSort(int arr[], int low, int high) {
    // 递归终止条件：当low >= high时，子数组只有一个或零个元素
    if (low < high) {
        // 获取分区索引
        int pi = partition(arr, low, high);
        // 递归排序基准左边的元素
        quickSort(arr, low, pi - 1);
        // 递归排序基准右边的元素
        quickSort(arr, pi + 1, high);
    }
}

// 辅助函数：打印数组
// 输入：arr-数组，size-数组大小
// 输出：无（打印到控制台）
void printArray(int arr[], int size) {
    for (int i = 0; i < size; i++) {
        printf("%d ", arr[i]);
    }
    printf("\n");
}

// 主函数：演示快速排序
int main() {
    // 初始化测试数组
    int arr[] = {10, 7, 8, 9, 1, 5};
    // 计算数组大小
    int n = sizeof(arr) / sizeof(arr[0]);
    
    printf("原始数组: ");
    printArray(arr, n);
    
    // 调用快速排序
    quickSort(arr, 0, n - 1);
    
    printf("排序后数组: ");
    printArray(arr, n);
    
    return 0;
}
```

### 注意事项

- **最坏情况**：当数组已经有序或所有元素相同时，时间复杂度退化为O(n²)。可以通过随机选择基准元素来缓解这个问题。
- **空间复杂度**：由于使用递归，空间复杂度为O(log n)（平均情况）到O(n)（最坏情况）。
- **不稳定排序**：快速排序不是稳定排序算法，相等元素的相对位置可能会改变。
- **小数组优化**：对于小规模数组（通常小于10个元素），插入排序可能比快速排序更快，可以在实现中加入这种优化。

快速排序通过分治策略高效地对数组进行排序，在平均情况下具有优秀的性能表现，是学习高效排序算法的重要起点。

## 17.2 归并排序

归并排序也是一种分治算法，它将数组分成两半，递归地对每一半进行排序，然后将两个已排序的半部分合并成一个完整的排序数组。归并排序的最大特点是它是稳定的排序算法，且时间复杂度始终为O(n log n)，不受输入数据的影响。

### 归并排序核心方法

| 功能名称       | 方法调用               | 具体功能与注意事项                                           |
| -------------- | ---------------------- | ------------------------------------------------------------ |
| 合并操作       | `merge(arr, l, m, r)`  | 合并两个已排序的子数组。必需参数：数组、左子数组起始索引、中间索引、右子数组结束索引 |
| 归并排序主函数 | `mergeSort(arr, l, r)` | 递归实现归并排序。必需参数：数组、起始索引、结束索引。注意递归终止条件 |
| 辅助数组管理   | `malloc/free`          | 归并过程需要额外的存储空间，必须正确分配和释放内存           |

### 使用示例

```c
#include <stdio.h>
#include <stdlib.h>

// 合并两个已排序的子数组
// 输入：arr-原数组，l-左子数组起始索引，m-中间索引，r-右子数组结束索引
// 输出：无（原地修改数组）
void merge(int arr[], int l, int m, int r) {
    // 计算左右子数组的大小
    int n1 = m - l + 1;
    int n2 = r - m;
    
    // 创建临时数组存储左右子数组
    int *L = (int*)malloc(n1 * sizeof(int));
    int *R = (int*)malloc(n2 * sizeof(int));
    
    // 复制数据到临时数组
    for (int i = 0; i < n1; i++) {
        L[i] = arr[l + i];  // 复制左子数组
    }
    for (int j = 0; j < n2; j++) {
        R[j] = arr[m + 1 + j];  // 复制右子数组
    }
    
    // 合并临时数组回到原数组
    int i = 0;    // 左子数组的初始索引
    int j = 0;    // 右子数组的初始索引
    int k = l;    // 合并后数组的初始索引
    
    // 比较左右子数组元素，将较小的放入原数组
    while (i < n1 && j < n2) {
        if (L[i] <= R[j]) {
            arr[k] = L[i];
            i++;
        } else {
            arr[k] = R[j];
            j++;
        }
        k++;
    }
    
    // 复制左子数组剩余元素（如果有）
    while (i < n1) {
        arr[k] = L[i];
        i++;
        k++;
    }
    
    // 复制右子数组剩余元素（如果有）
    while (j < n2) {
        arr[k] = R[j];
        j++;
        k++;
    }
    
    // 释放临时数组内存
    free(L);
    free(R);
}

// 归并排序主函数
// 输入：arr-待排序数组，l-起始索引，r-结束索引
// 输出：无（原地排序）
void mergeSort(int arr[], int l, int r) {
    // 递归终止条件：当l >= r时，子数组只有一个或零个元素
    if (l < r) {
        // 计算中间索引，避免溢出
        int m = l + (r - l) / 2;
        
        // 递归排序左半部分
        mergeSort(arr, l, m);
        // 递归排序右半部分
        mergeSort(arr, m + 1, r);
        // 合并两个已排序的部分
        merge(arr, l, m, r);
    }
}

// 辅助函数：打印数组
// 输入：arr-数组，size-数组大小
// 输出：无（打印到控制台）
void printArray(int arr[], int size) {
    for (int i = 0; i < size; i++) {
        printf("%d ", arr[i]);
    }
    printf("\n");
}

// 主函数：演示归并排序
int main() {
    // 初始化测试数组
    int arr[] = {12, 11, 13, 5, 6, 7};
    // 计算数组大小
    int arr_size = sizeof(arr) / sizeof(arr[0]);
    
    printf("原始数组: \n");
    printArray(arr, arr_size);
    
    // 调用归并排序
    mergeSort(arr, 0, arr_size - 1);
    
    printf("排序后数组: \n");
    printArray(arr, arr_size);
    
    return 0;
}
```

### 注意事项

- **空间复杂度**：归并排序需要O(n)的额外空间来存储临时数组，这是它的主要缺点。
- **稳定性**：归并排序是稳定的排序算法，相等元素的相对位置不会改变。
- **时间复杂度**：无论最好、平均还是最坏情况，时间复杂度都是O(n log n)。
- **自底向上实现**：除了递归实现外，归并排序也可以用迭代方式实现，避免递归调用栈的开销。
- **内存管理**：在C语言中必须手动管理内存，确保分配的临时数组被正确释放，避免内存泄漏。

归并排序以其稳定的时间复杂度和稳定性著称，特别适合对链表进行排序或需要稳定排序的场景。

## 17.3 堆排序

堆排序利用堆这种数据结构进行排序。堆是一个近似完全二叉树，满足堆性质：最大堆中父节点的值总是大于或等于其子节点的值；最小堆则相反。堆排序首先构建一个最大堆，然后反复将堆顶元素（最大值）与末尾元素交换，并调整堆，直到整个数组有序。

### 堆排序核心方法

| 功能名称     | 方法调用             | 具体功能与注意事项                                           |
| ------------ | -------------------- | ------------------------------------------------------------ |
| 堆调整       | `heapify(arr, n, i)` | 维护堆性质，确保以i为根的子树满足堆性质。必需参数：数组、堆大小、根节点索引 |
| 构建堆       | `buildHeap(arr, n)`  | 将普通数组转换为堆。必需参数：数组、数组大小                 |
| 堆排序主函数 | `heapSort(arr, n)`   | 实现堆排序算法。必需参数：数组、数组大小                     |

### 使用示例

```c
#include <stdio.h>

// 堆调整函数：维护最大堆性质
// 输入：arr-数组，n-堆大小，i-需要调整的根节点索引
// 输出：无（原地调整堆）
void heapify(int arr[], int n, int i) {
    // 初始化最大值为根节点
    int largest = i;
    // 计算左子节点索引
    int left = 2 * i + 1;
    // 计算右子节点索引
    int right = 2 * i + 2;
    
    // 如果左子节点存在且大于根节点
    if (left < n && arr[left] > arr[largest]) {
        largest = left;
    }
    
    // 如果右子节点存在且大于当前最大值
    if (right < n && arr[right] > arr[largest]) {
        largest = right;
    }
    
    // 如果最大值不是根节点
    if (largest != i) {
        // 交换根节点和最大值
        int temp = arr[i];
        arr[i] = arr[largest];
        arr[largest] = temp;
        
        // 递归调整受影响的子树
        heapify(arr, n, largest);
    }
}

// 构建最大堆
// 输入：arr-数组，n-数组大小
// 输出：无（原地构建堆）
void buildHeap(int arr[], int n) {
    // 从最后一个非叶子节点开始向上调整
    // 最后一个非叶子节点索引为(n/2)-1
    for (int i = n / 2 - 1; i >= 0; i--) {
        heapify(arr, n, i);
    }
}

// 堆排序主函数
// 输入：arr-待排序数组，n-数组大小
// 输出：无（原地排序）
void heapSort(int arr[], int n) {
    // 构建最大堆
    buildHeap(arr, n);
    
    // 逐个提取元素
    for (int i = n - 1; i > 0; i--) {
        // 将当前最大值（堆顶）移到末尾
        int temp = arr[0];
        arr[0] = arr[i];
        arr[i] = temp;
        
        // 对减少后的堆进行调整
        heapify(arr, i, 0);
    }
}

// 辅助函数：打印数组
// 输入：arr-数组，size-数组大小
// 输出：无（打印到控制台）
void printArray(int arr[], int size) {
    for (int i = 0; i < size; i++) {
        printf("%d ", arr[i]);
    }
    printf("\n");
}

// 主函数：演示堆排序
int main() {
    // 初始化测试数组
    int arr[] = {12, 11, 13, 5, 6, 7};
    // 计算数组大小
    int n = sizeof(arr) / sizeof(arr[0]);
    
    printf("原始数组: \n");
    printArray(arr, n);
    
    // 调用堆排序
    heapSort(arr, n);
    
    printf("排序后数组: \n");
    printArray(arr, n);
    
    return 0;
}
```

### 注意事项

- **原地排序**：堆排序是原地排序算法，只需要O(1)的额外空间。
- **时间复杂度**：最好、平均和最坏情况下的时间复杂度都是O(n log n)。
- **不稳定排序**：堆排序不是稳定排序算法，相等元素的相对位置可能会改变。
- **缓存性能**：由于堆排序的访问模式不是局部的，缓存性能通常不如快速排序。
- **应用场景**：堆排序常用于实现优先队列，或者在需要保证最坏情况时间复杂度的场景中使用。

堆排序结合了插入排序的空间效率和归并排序的时间效率，是一种可靠的排序算法，特别适合内存受限的环境。
