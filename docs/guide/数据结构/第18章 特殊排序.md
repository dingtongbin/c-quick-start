### 18.1 计数排序

计数排序是一种非比较型整数排序算法，其原理是将输入的数据值转化为键存储在额外开辟的数组空间中。作为一种线性时间复杂度的排序，计数排序要求输入的数据必须是有确定范围的整数。

| 功能名称       | 实例调用方法                                    | 具体功能与注意事项                                           |
| -------------- | ----------------------------------------------- | ------------------------------------------------------------ |
| 计数排序实现   | `counting_sort(arr, size, max_val)`             | 对数组进行计数排序，max_val为数组中最大值，需要额外O(max_val)空间 |
| 辅助数组初始化 | `memset(count, 0, sizeof(int) * (max_val + 1))` | 初始化计数数组，确保所有元素为0                              |

```c
#include <stdio.h>
#include <stdlib.h>
#include <string.h>

// 计数排序函数
// arr: 待排序数组
// size: 数组大小
// max_val: 数组中的最大值
void counting_sort(int arr[], int size, int max_val) {
    // 创建计数数组，大小为max_val+1
    int *count = (int*)malloc(sizeof(int) * (max_val + 1));
    
    // 初始化计数数组为0
    memset(count, 0, sizeof(int) * (max_val + 1));
    
    // 统计每个元素出现的次数
    for (int i = 0; i < size; i++) {
        count[arr[i]]++;
    }
    
    // 重构原数组
    int index = 0;
    for (int i = 0; i <= max_val; i++) {
        // 将每个值按出现次数放回原数组
        while (count[i] > 0) {
            arr[index++] = i;
            count[i]--;
        }
    }
    
    // 释放动态分配的内存
    free(count);
}

// 测试计数排序
int main() {
    int arr[] = {4, 2, 2, 8, 3, 3, 1};
    int size = sizeof(arr) / sizeof(arr[0]);
    int max_val = 8; // 数组中的最大值
    
    printf("排序前: ");
    for (int i = 0; i < size; i++) {
        printf("%d ", arr[i]);
    }
    printf("\n");
    
    // 调用计数排序
    counting_sort(arr, size, max_val);
    
    printf("排序后: ");
    for (int i = 0; i < size; i++) {
        printf("%d ", arr[i]);
    }
    printf("\n");
    
    return 0;
}
```

**注意事项：**

1. 计数排序只适用于整数排序，且数据范围不能太大，否则会浪费大量空间
2. 需要预先知道数据的最大值，或者先遍历一次数组找到最大值
3. 计数排序是稳定的排序算法
4. 对于负数需要特殊处理，通常通过偏移量转换为正数

计数排序通过统计每个元素的出现次数来实现排序，时间复杂度为线性，适用于数据范围较小的整数排序场景。

### 18.2 桶排序

桶排序是计数排序的升级版，它利用了函数的映射关系，高效与否的关键就在于这个映射函数的确定。桶排序的工作原理是将数组分到有限数量的桶里，每个桶再分别排序（有可能再使用别的排序算法或是以递归方式继续使用桶排序）。

| 功能名称     | 实例调用方法                          | 具体功能与注意事项                     |
| ------------ | ------------------------------------- | -------------------------------------- |
| 桶排序实现   | `bucket_sort(arr, size)`              | 对浮点数数组进行桶排序，默认使用10个桶 |
| 插入排序辅助 | `insertion_sort(bucket, bucket_size)` | 对单个桶内的元素进行插入排序           |

```c
#include <stdio.h>
#include <stdlib.h>
#include <string.h>

// 插入排序函数，用于对单个桶排序
// arr: 待排序数组
// size: 数组大小
void insertion_sort(float arr[], int size) {
    for (int i = 1; i < size; i++) {
        float key = arr[i];
        int j = i - 1;
        
        // 将大于key的元素向后移动
        while (j >= 0 && arr[j] > key) {
            arr[j + 1] = arr[j];
            j--;
        }
        arr[j + 1] = key;
    }
}

// 桶排序函数
// arr: 待排序的浮点数数组
// size: 数组大小
void bucket_sort(float arr[], int size) {
    // 创建桶数组，这里使用10个桶
    const int bucket_count = 10;
    float **buckets = (float**)malloc(sizeof(float*) * bucket_count);
    int *bucket_sizes = (int*)calloc(bucket_count, sizeof(int));
    
    // 初始化每个桶
    for (int i = 0; i < bucket_count; i++) {
        buckets[i] = (float*)malloc(sizeof(float) * size);
    }
    
    // 将元素分配到各个桶中
    // 假设所有元素都在[0, 1)范围内
    for (int i = 0; i < size; i++) {
        int bucket_index = (int)(arr[i] * bucket_count);
        // 处理边界情况，确保索引不越界
        if (bucket_index >= bucket_count) {
            bucket_index = bucket_count - 1;
        }
        buckets[bucket_index][bucket_sizes[bucket_index]] = arr[i];
        bucket_sizes[bucket_index]++;
    }
    
    // 对每个非空桶进行排序
    int index = 0;
    for (int i = 0; i < bucket_count; i++) {
        if (bucket_sizes[i] > 0) {
            insertion_sort(buckets[i], bucket_sizes[i]);
            // 将排序后的桶元素复制回原数组
            for (int j = 0; j < bucket_sizes[i]; j++) {
                arr[index++] = buckets[i][j];
            }
        }
    }
    
    // 释放内存
    for (int i = 0; i < bucket_count; i++) {
        free(buckets[i]);
    }
    free(buckets);
    free(bucket_sizes);
}

// 测试桶排序
int main() {
    float arr[] = {0.42, 0.32, 0.33, 0.52, 0.37, 0.47, 0.51};
    int size = sizeof(arr) / sizeof(arr[0]);
    
    printf("排序前: ");
    for (int i = 0; i < size; i++) {
        printf("%.2f ", arr[i]);
    }
    printf("\n");
    
    // 调用桶排序
    bucket_sort(arr, size);
    
    printf("排序后: ");
    for (int i = 0; i < size; i++) {
        printf("%.2f ", arr[i]);
    }
    printf("\n");
    
    return 0;
}
```

**注意事项：**

1. 桶排序假设输入数据是均匀分布的，这样能保证平均时间复杂度为O(n)
2. 桶的数量选择很重要，通常根据数据范围和分布来确定
3. 如果数据分布不均匀，可能导致某些桶包含大量元素，降低效率
4. 桶排序不是比较排序，不受O(n log n)下限限制

桶排序通过将数据分桶后再排序，充分利用了数据分布特性，在数据均匀分布时能达到线性时间复杂度。

### 18.3 基数排序

基数排序是一种非比较型整数排序算法，其原理是将整数按位数切割成不同的数字，然后按每个位数分别比较。基数排序可以采用LSD（Least Significant Digit，最低位优先）或MSD（Most Significant Digit，最高位优先）两种方式。

| 功能名称     | 实例调用方法                              | 具体功能与注意事项                 |
| ------------ | ----------------------------------------- | ---------------------------------- |
| 基数排序实现 | `radix_sort(arr, size)`                   | 对整数数组进行基数排序，使用10进制 |
| 计数排序辅助 | `counting_sort_for_radix(arr, size, exp)` | 按指定位进行计数排序               |

```c
#include <stdio.h>
#include <stdlib.h>
#include <string.h>

// 获取数组中的最大值
// arr: 输入数组
// size: 数组大小
int get_max(int arr[], int size) {
    int max = arr[0];
    for (int i = 1; i < size; i++) {
        if (arr[i] > max) {
            max = arr[i];
        }
    }
    return max;
}

// 基数排序中使用的计数排序（按指定位）
// arr: 待排序数组
// size: 数组大小
// exp: 当前位的权重（1, 10, 100...）
void counting_sort_for_radix(int arr[], int size, int exp) {
    // 输出数组
    int *output = (int*)malloc(sizeof(int) * size);
    // 计数数组，基数为10（0-9）
    int count[10] = {0};
    
    // 统计当前位上各数字的出现次数
    for (int i = 0; i < size; i++) {
        count[(arr[i] / exp) % 10]++;
    }
    
    // 计算累积计数，得到每个数字在输出数组中的位置
    for (int i = 1; i < 10; i++) {
        count[i] += count[i - 1];
    }
    
    // 从后往前构建输出数组，保证稳定性
    for (int i = size - 1; i >= 0; i--) {
        output[count[(arr[i] / exp) % 10] - 1] = arr[i];
        count[(arr[i] / exp) % 10]--;
    }
    
    // 将输出数组复制回原数组
    for (int i = 0; i < size; i++) {
        arr[i] = output[i];
    }
    
    // 释放内存
    free(output);
}

// 基数排序主函数
// arr: 待排序数组
// size: 数组大小
void radix_sort(int arr[], int size) {
    // 找到最大值，确定需要处理的位数
    int max = get_max(arr, size);
    
    // 对每一位进行计数排序
    for (int exp = 1; max / exp > 0; exp *= 10) {
        counting_sort_for_radix(arr, size, exp);
    }
}

// 测试基数排序
int main() {
    int arr[] = {170, 45, 75, 90, 2, 802, 24, 66};
    int size = sizeof(arr) / sizeof(arr[0]);
    
    printf("排序前: ");
    for (int i = 0; i < size; i++) {
        printf("%d ", arr[i]);
    }
    printf("\n");
    
    // 调用基数排序
    radix_sort(arr, size);
    
    printf("排序后: ");
    for (int i = 0; i < size; i++) {
        printf("%d ", arr[i]);
    }
    printf("\n");
    
    return 0;
}
```

**注意事项：**

1. 基数排序只能用于整数或可以转换为整数的数据类型
2. 基数排序是稳定排序，这一点在实现中通过从后往前遍历保证
3. 时间复杂度取决于位数，对于固定位数的整数，可以认为是线性时间
4. 基数排序的空间复杂度较高，需要额外的输出数组和计数数组

基数排序通过逐位排序的方式实现整体有序，利用了计数排序的稳定性，适合处理位数相对较少的整数排序问题。
