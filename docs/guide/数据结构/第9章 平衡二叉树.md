### 9.1 AVL树概念

AVL树是一种自平衡的二叉搜索树，它的名字来源于两位发明者Adelson-Velsky和Landis。在AVL树中，任何节点的两个子树的高度差最多为1，这种特性确保了树的操作（如查找、插入和删除）的时间复杂度保持在O(log n)。

| 版本信息 | 描述                                                         |
| -------- | ------------------------------------------------------------ |
| C标准    | C99或更高                                                    |
| 依赖库   | 无特殊依赖                                                   |
| 适用场景 | 需要频繁进行查找、插入和删除操作，并且要求这些操作具有对数时间复杂度的应用 |

**实例方法表格**

| 功能名称      | 实例调用方法                       | 具体功能、注意事项、必需参数/可选参数                        |
| ------------- | ---------------------------------- | ------------------------------------------------------------ |
| 创建AVL树节点 | `AVLNode* createNode(int key)`     | 创建一个新的AVL树节点；必需参数：key - 节点的关键字值        |
| 获取节点高度  | `int getHeight(AVLNode* node)`     | 返回给定节点的高度；必需参数：node - 指向AVL树节点的指针     |
| 更新节点高度  | `void updateHeight(AVLNode* node)` | 根据左右子树更新当前节点的高度；必需参数：node - 指向AVL树节点的指针 |

下面是一个简单的创建AVL树节点的例子：

```c
#include <stdio.h>
#include <stdlib.h>

// 定义AVL树节点结构
typedef struct AVLNode {
    int key; // 关键字
    struct AVLNode *left; // 左子树
    struct AVLNode *right; // 右子树
    int height; // 节点高度
} AVLNode;

// 创建新的AVL树节点
AVLNode* createNode(int key) {
    // 分配内存给新节点
    AVLNode* node = (AVLNode*)malloc(sizeof(AVLNode));
    if (!node) { // 如果分配失败
        printf("内存分配失败\n");
        return NULL;
    }
    node->key = key; // 设置关键字
    node->left = NULL; // 初始化左子树为空
    node->right = NULL; // 初始化右子树为空
    node->height = 1; // 新节点的高度初始化为1
    return node; // 返回新创建的节点
}

// 获取节点的高度
int getHeight(AVLNode* node) {
    if (node == NULL) // 如果节点为空，则高度为0
        return 0;
    return node->height; // 否则返回节点的实际高度
}

// 更新节点的高度
void updateHeight(AVLNode* node) {
    if (node != NULL) { // 确保节点非空
        // 节点的高度等于其左右子树最大高度加1
        node->height = 1 + (getHeight(node->left) > getHeight(node->right) ? getHeight(node->left) : getHeight(node->right));
    }
}
```

**注意事项**

- 在处理AVL树时，每次插入或删除后都需要检查并可能调整树以维持平衡。
- 节点的高度是基于其子树的最大深度来计算的，因此在修改树结构后需要更新相关节点的高度。

这一节我们介绍了AVL树的基本概念以及如何创建AVL树节点和管理节点的高度。理解这些基础知识对于后续学习AVL树的旋转操作至关重要。

### 9.2 旋转操作

为了保持AVL树的平衡性，在插入或删除节点之后，如果发现某个节点不再满足AVL性质（即左右子树高度差超过1），就需要通过旋转操作来恢复平衡。主要的旋转类型有四种：单左旋(LL)、单右旋(RR)、左右双旋(LR)和右左双旋(RL)。

| 功能名称 | 实例调用方法                          | 具体功能、注意事项、必需参数/可选参数                      |
| -------- | ------------------------------------- | ---------------------------------------------------------- |
| 单右旋   | `AVLNode* rotateRight(AVLNode* y)`    | 对于不平衡的节点y执行单右旋；必需参数：y - 不平衡节点      |
| 单左旋   | `AVLNode* rotateLeft(AVLNode* x)`     | 对于不平衡的节点x执行单左旋；必需参数：x - 不平衡节点      |
| 左右双旋 | `AVLNode* doubleRotateLR(AVLNode* z)` | 先对z的左孩子做左旋，再对z做右旋；必需参数：z - 不平衡节点 |
| 右左双旋 | `AVLNode* doubleRotateRL(AVLNode* z)` | 先对z的右孩子做右旋，再对z做左旋；必需参数：z - 不平衡节点 |

这里给出一个单右旋的例子：

```c
// 单右旋
AVLNode* rotateRight(AVLNode* y) {
    AVLNode* x = y->left; // x成为新的根
    AVLNode* T2 = x->right; // T2是x的右子树

    // 执行旋转
    x->right = y;
    y->left = T2;

    // 更新高度
    updateHeight(y); // 先更新较低的节点
    updateHeight(x); // 再更新较高的节点

    // 返回新的根
    return x;
}
```

**注意事项**

- 旋转操作必须正确地重新连接子树，并且更新涉及节点的高度。
- 在实际应用中，通常会结合获取平衡因子的方法来决定何时及何种类型的旋转应该被执行。

这节讲解了AVL树中的旋转操作，这是维持AVL树平衡的关键步骤。掌握这些旋转技巧能够帮助我们在插入或删除元素后有效地维护AVL树的结构。

### 9.3 AVL树的插入

在AVL树中插入一个新节点的过程类似于普通二叉搜索树的插入过程，但在插入完成后，我们需要从插入点向上回溯到根节点，沿途检查每个节点是否仍然保持AVL性质。如果不满足，则需要进行相应的旋转操作来恢复平衡。

| 功能名称 | 实例调用方法                              | 具体功能、注意事项、必需参数/可选参数                        |
| -------- | ----------------------------------------- | ------------------------------------------------------------ |
| 插入节点 | `AVLNode* insert(AVLNode* node, int key)` | 在AVL树中插入关键字为key的新节点；必需参数：node - 当前子树的根节点，key - 要插入的关键字 |

下面是AVL树插入操作的一个示例代码：

```c
// 计算平衡因子
int getBalance(AVLNode* node) {
    if (node == NULL)
        return 0;
    return getHeight(node->left) - getHeight(node->right);
}

// 插入节点
AVLNode* insert(AVLNode* node, int key) {
    // 1. 执行正常的BST插入
    if (node == NULL)
        return(createNode(key));

    if (key < node->key)
        node->left = insert(node->left, key);
    else if (key > node->key)
        node->right = insert(node->right, key);
    else // 相同的key不允许重复插入
        return node;

    // 2. 更新此祖先节点的高度
    updateHeight(node);

    // 3. 获取该节点的平衡因子
    int balance = getBalance(node);

    // 如果这个节点变得不平衡，那么就有4种情况
    // Left Left Case
    if (balance > 1 && key < node->left->key)
        return rotateRight(node);

    // Right Right Case
    if (balance < -1 && key > node->right->key)
        return rotateLeft(node);

    // Left Right Case
    if (balance > 1 && key > node->left->key) {
        node->left = rotateLeft(node->left);
        return rotateRight(node);
    }

    // Right Left Case
    if (balance < -1 && key < node->right->key) {
        node->right = rotateRight(node->right);
        return rotateLeft(node);
    }

    // 返回未改变的节点指针
    return node;
}
```

**注意事项**

- 插入过程中要注意不要重复插入相同的键值。
- 插入后必须沿着路径向上更新所有受影响节点的高度，并根据需要执行适当的旋转操作。

本节详细说明了如何在AVL树中安全地插入新节点，同时保证树的平衡性。通过这种方式，我们可以确保AVL树始终提供高效的查找性能。

### 9.4 AVL树的删除

删除AVL树中的节点比插入更复杂一些，因为除了要找到待删除的节点外，还需要考虑三种不同的情况：删除叶子节点、删除只有一个孩子的节点、删除有两个孩子的节点。与插入类似，删除后也需要检查并修复可能破坏的AVL性质。

| 功能名称 | 实例调用方法                                  | 具体功能、注意事项、必需参数/可选参数                        |
| -------- | --------------------------------------------- | ------------------------------------------------------------ |
| 删除节点 | `AVLNode* deleteNode(AVLNode* root, int key)` | 从AVL树中删除关键字为key的节点；必需参数：root - AVL树的根节点，key - 要删除的关键字 |

以下是一个删除操作的简化版本：

```c
// 查找最小值节点
AVLNode* minValueNode(AVLNode* node) {
    AVLNode* current = node;
    while (current->left != NULL)
        current = current->left;
    return current;
}

// 删除节点
AVLNode* deleteNode(AVLNode* root, int key) {
    // STEP 1: PERFORM STANDARD BST DELETE
    if (root == NULL)
        return root;

    if (key < root->key)
        root->left = deleteNode(root->left, key);
    else if (key > root->key)
        root->right = deleteNode(root->right, key);
    else {
        // node with only one child or no child
        if ((root->left == NULL) || (root->right == NULL)) {
            AVLNode* temp = root->left ? root->left : root->right;

            // No child case
            if (temp == NULL) {
                temp = root;
                root = NULL;
            } else // One child case
                *root = *temp; // Copy the contents of the non-empty child

            free(temp);
        } else {
            // node with two children: Get the inorder successor (smallest
            // in the right subtree)
            AVLNode* temp = minValueNode(root->right);

            // Copy the inorder successor's data to this node
            root->key = temp->key;

            // Delete the inorder successor
            root->right = deleteNode(root->right, temp->key);
        }
    }

    // If the tree had only one node then return
    if (root == NULL)
        return root;

    // STEP 2: UPDATE HEIGHT OF THE CURRENT NODE
    updateHeight(root);

    // STEP 3: GET THE BALANCE FACTOR OF THIS NODE (to check whether
    // this node became unbalanced)
    int balance = getBalance(root);

    // If this node becomes unbalanced, then there are 4 cases

    // Left Left Case
    if (balance > 1 && getBalance(root->left) >= 0)
        return rotateRight(root);

    // Left Right Case
    if (balance > 1 && getBalance(root->left) < 0) {
        root->left = rotateLeft(root->left);
        return rotateRight(root);
    }

    // Right Right Case
    if (balance < -1 && getBalance(root->right) <= 0)
        return rotateLeft(root);

    // Right Left Case
    if (balance < -1 && getBalance(root->right) > 0) {
        root->right = rotateRight(root->right);
        return rotateLeft(root);
    }

    return root;
}
```

**注意事项**

- 删除有两个孩子的节点时，通常使用其中序后继（右子树中的最小值）或中序前驱（左子树中的最大值）来替换被删除的节点。
- 删除操作后同样需要沿路径向上更新高度，并且根据平衡因子判断是否需要旋转。

至此，我们已经完成了AVL树删除操作的学习。通过合理运用前面学到的旋转技术，即使在删除节点后也能保持AVL树的良好平衡状态。
