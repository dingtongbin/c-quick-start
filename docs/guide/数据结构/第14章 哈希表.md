### 14.1 哈希表概念

哈希表（Hash Table）是一种高效的数据结构，它通过哈希函数将键（key）映射到表中的一个位置来访问记录，以加快查找速度。理想情况下，哈希表的插入、删除和查找操作的时间复杂度都是 O(1)。

| 技术点          | 版本/环境要求 | 适用场景                                             |
| --------------- | ------------- | ---------------------------------------------------- |
| C语言哈希表实现 | C99标准及以上 | 需要快速查找、插入、删除的场景，如缓存、数据库索引等 |

**哈希表核心概念：**

- **哈希函数**：将任意大小的数据映射到固定大小的值
- **哈希冲突**：不同的键映射到相同的哈希值
- **负载因子**：已存储元素数量与哈希表容量的比值

哈希表的基本思想就是用空间换时间，通过合理的哈希函数设计，让数据分布尽可能均匀，从而实现高效的查找性能。

### 14.2 哈希函数设计

哈希函数的设计是哈希表性能的关键。一个好的哈希函数应该具备以下特点：

1. 计算简单快速
2. 哈希值分布均匀
3. 相同输入始终产生相同输出

| 哈希函数方法 | 实现方式                   | 特点说明                     |
| ------------ | -------------------------- | ---------------------------- |
| 除留余数法   | key % table_size           | 最常用，table_size最好为质数 |
| 乘法哈希     | (key * A) % 1 * table_size | A通常取(√5-1)/2 ≈ 0.618      |
| 平方取中法   | 取key平方的中间几位        | 适用于key分布不均匀的情况    |

```c
// 除留余数法哈希函数实现
#include <stdio.h>
#include <stdlib.h>

// 哈希函数：除留余数法
// 参数：key - 要哈希的键值，size - 哈希表大小
// 返回值：哈希后的索引位置
int hash_function(int key, int size) {
    // 使用绝对值确保负数也能正确处理
    return abs(key) % size;
}

// 乘法哈希函数实现
// 参数：key - 要哈希的键值，size - 哈希表大小
// 返回值：哈希后的索引位置
int hash_function_multiplication(int key, int size) {
    // 黄金比例常数
    const double A = 0.6180339887;
    // 先取绝对值避免负数问题
    int abs_key = abs(key);
    // 计算 key * A 的小数部分，再乘以表大小
    double fractional_part = (abs_key * A) - (int)(abs_key * A);
    return (int)(fractional_part * size);
}

// 测试哈希函数
int main() {
    int keys[] = {10, 22, 31, 4, 15, 28, 17, 88, 59};
    int num_keys = sizeof(keys) / sizeof(keys[0]);
    int table_size = 11; // 使用质数作为表大小
    
    printf("除留余数法哈希结果:\n");
    for (int i = 0; i < num_keys; i++) {
        int hash_index = hash_function(keys[i], table_size);
        printf("key=%d -> hash=%d\n", keys[i], hash_index);
    }
    
    printf("\n乘法哈希结果:\n");
    for (int i = 0; i < num_keys; i++) {
        int hash_index = hash_function_multiplication(keys[i], table_size);
        printf("key=%d -> hash=%d\n", keys[i], hash_index);
    }
    
    return 0;
}
```

**注意事项：**

- 哈希表大小选择很重要，质数可以减少冲突
- 对于字符串键，需要先转换为整数再哈希
- 哈希函数必须保证相同的输入总是产生相同的输出

好的哈希函数能让数据在哈希表中均匀分布，大大减少冲突的发生，从而保证哈希表的高效性能。

### 14.3 冲突处理方法

即使有了好的哈希函数，冲突仍然不可避免。常见的冲突处理方法有：

| 冲突处理方法 | 实现方式           | 优缺点                                       |
| ------------ | ------------------ | -------------------------------------------- |
| 链地址法     | 每个桶维护一个链表 | 简单易实现，内存利用率高，但需要额外指针开销 |
| 开放地址法   | 寻找下一个空闲位置 | 无需额外指针，但容易产生聚集现象             |
| 再哈希法     | 使用第二个哈希函数 | 减少聚集，但计算开销大                       |

**链地址法（Separate Chaining）：**
每个哈希表的槽位都指向一个链表，所有哈希到同一位置的元素都存储在这个链表中。

**开放地址法（Open Addressing）：**
当发生冲突时，在哈希表中寻找下一个可用的位置。常见的探测方法有：

- 线性探测：h(k,i) = (h(k) + i) % m
- 二次探测：h(k,i) = (h(k) + c1*i + c2*i²) % m  
- 双重哈希：h(k,i) = (h1(k) + i*h2(k)) % m

```c
// 链地址法冲突处理示例
#include <stdio.h>
#include <stdlib.h>
#include <string.h>

// 链表节点结构
typedef struct Node {
    int key;
    char* value;
    struct Node* next;
} Node;

// 创建新节点
// 参数：key - 键，value - 值
// 返回值：新创建的节点指针
Node* create_node(int key, const char* value) {
    Node* new_node = (Node*)malloc(sizeof(Node));
    if (!new_node) {
        fprintf(stderr, "内存分配失败\n");
        return NULL;
    }
    new_node->key = key;
    new_node->value = strdup(value); // 复制字符串
    new_node->next = NULL;
    return new_node;
}

// 在链表头部插入节点
// 参数：head - 链表头指针，new_node - 要插入的节点
// 返回值：新的链表头指针
Node* insert_at_head(Node* head, Node* new_node) {
    if (!new_node) return head;
    new_node->next = head;
    return new_node;
}

// 在链表中查找指定键
// 参数：head - 链表头指针，key - 要查找的键
// 返回值：找到的节点指针，未找到返回NULL
Node* search_in_chain(Node* head, int key) {
    Node* current = head;
    while (current != NULL) {
        if (current->key == key) {
            return current;
        }
        current = current->next;
    }
    return NULL;
}

// 释放链表内存
// 参数：head - 链表头指针
void free_chain(Node* head) {
    while (head != NULL) {
        Node* temp = head;
        head = head->next;
        free(temp->value);
        free(temp);
    }
}

// 测试链地址法
int main() {
    // 模拟哈希表，大小为5
    Node* hash_table[5] = {NULL};
    
    // 插入一些数据
    int keys[] = {10, 15, 20, 25, 30};
    char* values[] = {"ten", "fifteen", "twenty", "twenty-five", "thirty"};
    int num_items = 5;
    
    // 使用简单的哈希函数：key % 5
    for (int i = 0; i < num_items; i++) {
        int hash_index = keys[i] % 5;
        Node* new_node = create_node(keys[i], values[i]);
        if (new_node) {
            hash_table[hash_index] = insert_at_head(hash_table[hash_index], new_node);
            printf("插入: key=%d, value=%s, hash_index=%d\n", 
                   keys[i], values[i], hash_index);
        }
    }
    
    // 查找测试
    printf("\n查找测试:\n");
    int search_keys[] = {15, 22, 25};
    for (int i = 0; i < 3; i++) {
        int hash_index = search_keys[i] % 5;
        Node* found = search_in_chain(hash_table[hash_index], search_keys[i]);
        if (found) {
            printf("找到 key=%d, value=%s\n", found->key, found->value);
        } else {
            printf("未找到 key=%d\n", search_keys[i]);
        }
    }
    
    // 清理内存
    for (int i = 0; i < 5; i++) {
        free_chain(hash_table[i]);
    }
    
    return 0;
}
```

**注意事项：**

- 链地址法在最坏情况下退化为链表，时间复杂度变为O(n)
- 开放地址法需要考虑删除操作的特殊处理（不能简单置空）
- 负载因子超过0.7时应该考虑扩容

冲突处理方法的选择直接影响哈希表的性能，需要根据具体应用场景来决定使用哪种方法。

### 14.4 哈希表实现

现在我们来实现一个完整的哈希表，包含基本的插入、查找、删除操作。

| 哈希表操作   | 方法调用                             | 功能说明                 |
| ------------ | ------------------------------------ | ------------------------ |
| 初始化哈希表 | hash_table_init(size)                | 创建指定大小的哈希表     |
| 插入键值对   | hash_table_insert(table, key, value) | 插入新的键值对           |
| 查找值       | hash_table_search(table, key)        | 根据键查找对应的值       |
| 删除键值对   | hash_table_delete(table, key)        | 删除指定键的键值对       |
| 释放内存     | hash_table_free(table)               | 释放哈希表占用的所有内存 |

```c
// 完整的哈希表实现（链地址法）
#include <stdio.h>
#include <stdlib.h>
#include <string.h>
#include <stdbool.h>

// 哈希表节点
typedef struct HashNode {
    int key;
    char* value;
    struct HashNode* next;
} HashNode;

// 哈希表结构
typedef struct {
    HashNode** buckets;  // 桶数组
    int size;           // 哈希表大小
    int count;          // 当前元素数量
} HashTable;

// 哈希函数
// 参数：key - 键值，size - 哈希表大小
// 返回值：哈希索引
int hash_function(int key, int size) {
    return abs(key) % size;
}

// 初始化哈希表
// 参数：size - 哈希表初始大小
// 返回值：初始化后的哈希表指针，失败返回NULL
HashTable* hash_table_init(int size) {
    if (size <= 0) {
        fprintf(stderr, "哈希表大小必须大于0\n");
        return NULL;
    }
    
    HashTable* table = (HashTable*)malloc(sizeof(HashTable));
    if (!table) {
        fprintf(stderr, "内存分配失败\n");
        return NULL;
    }
    
    table->buckets = (HashNode**)calloc(size, sizeof(HashNode*));
    if (!table->buckets) {
        fprintf(stderr, "桶数组内存分配失败\n");
        free(table);
        return NULL;
    }
    
    table->size = size;
    table->count = 0;
    return table;
}

// 插入键值对
// 参数：table - 哈希表指针，key - 键，value - 值
// 返回值：成功返回true，失败返回false
bool hash_table_insert(HashTable* table, int key, const char* value) {
    if (!table || !value) {
        return false;
    }
    
    int index = hash_function(key, table->size);
    
    // 检查是否已存在相同的key
    HashNode* current = table->buckets[index];
    while (current != NULL) {
        if (current->key == key) {
            // 更新现有值
            free(current->value);
            current->value = strdup(value);
            return true;
        }
        current = current->next;
    }
    
    // 创建新节点
    HashNode* new_node = (HashNode*)malloc(sizeof(HashNode));
    if (!new_node) {
        return false;
    }
    
    new_node->key = key;
    new_node->value = strdup(value);
    if (!new_node->value) {
        free(new_node);
        return false;
    }
    
    // 插入到链表头部
    new_node->next = table->buckets[index];
    table->buckets[index] = new_node;
    table->count++;
    
    return true;
}

// 查找值
// 参数：table - 哈希表指针，key - 要查找的键
// 返回值：找到的值字符串，未找到返回NULL
char* hash_table_search(HashTable* table, int key) {
    if (!table) {
        return NULL;
    }
    
    int index = hash_function(key, table->size);
    HashNode* current = table->buckets[index];
    
    while (current != NULL) {
        if (current->key == key) {
            return current->value;
        }
        current = current->next;
    }
    
    return NULL;
}

// 删除键值对
// 参数：table - 哈希表指针，key - 要删除的键
// 返回值：成功返回true，失败返回false
bool hash_table_delete(HashTable* table, int key) {
    if (!table) {
        return false;
    }
    
    int index = hash_function(key, table->size);
    HashNode* current = table->buckets[index];
    HashNode* prev = NULL;
    
    while (current != NULL) {
        if (current->key == key) {
            // 找到要删除的节点
            if (prev == NULL) {
                // 删除头节点
                table->buckets[index] = current->next;
            } else {
                // 删除中间或尾部节点
                prev->next = current->next;
            }
            
            free(current->value);
            free(current);
            table->count--;
            return true;
        }
        prev = current;
        current = current->next;
    }
    
    return false; // 未找到
}

// 释放哈希表内存
// 参数：table - 哈希表指针
void hash_table_free(HashTable* table) {
    if (!table) return;
    
    for (int i = 0; i < table->size; i++) {
        HashNode* current = table->buckets[i];
        while (current != NULL) {
            HashNode* temp = current;
            current = current->next;
            free(temp->value);
            free(temp);
        }
    }
    
    free(table->buckets);
    free(table);
}

// 打印哈希表内容（用于调试）
// 参数：table - 哈希表指针
void hash_table_print(HashTable* table) {
    if (!table) return;
    
    printf("哈希表内容 (size=%d, count=%d):\n", table->size, table->count);
    for (int i = 0; i < table->size; i++) {
        if (table->buckets[i] != NULL) {
            printf("Bucket %d: ", i);
            HashNode* current = table->buckets[i];
            while (current != NULL) {
                printf("(%d, %s) -> ", current->key, current->value);
                current = current->next;
            }
            printf("NULL\n");
        }
    }
}

// 测试完整的哈希表实现
int main() {
    // 初始化哈希表
    HashTable* table = hash_table_init(7); // 使用质数7
    if (!table) {
        fprintf(stderr, "哈希表初始化失败\n");
        return 1;
    }
    
    // 插入数据
    printf("=== 插入数据 ===\n");
    hash_table_insert(table, 10, "apple");
    hash_table_insert(table, 20, "banana");
    hash_table_insert(table, 30, "cherry");
    hash_table_insert(table, 15, "date");
    hash_table_insert(table, 25, "elderberry");
    
    hash_table_print(table);
    
    // 查找测试
    printf("\n=== 查找测试 ===\n");
    char* result = hash_table_search(table, 20);
    if (result) {
        printf("找到 key=20: %s\n", result);
    } else {
        printf("未找到 key=20\n");
    }
    
    result = hash_table_search(table, 99);
    if (result) {
        printf("找到 key=99: %s\n", result);
    } else {
        printf("未找到 key=99\n");
    }
    
    // 删除测试
    printf("\n=== 删除测试 ===\n");
    if (hash_table_delete(table, 20)) {
        printf("成功删除 key=20\n");
    } else {
        printf("删除 key=20 失败\n");
    }
    
    if (hash_table_delete(table, 99)) {
        printf("成功删除 key=99\n");
    } else {
        printf("删除 key=99 失败（正常，因为不存在）\n");
    }
    
    hash_table_print(table);
    
    // 清理内存
    hash_table_free(table);
    printf("\n哈希表内存已释放\n");
    
    return 0;
}
```

**注意事项：**

- 哈希表需要动态扩容以维持良好的性能（负载因子 > 0.7时）
- 字符串值需要深拷贝，避免外部修改影响哈希表内部数据
- 删除操作需要正确处理链表的连接，避免内存泄漏
- 所有分配的内存都必须正确释放，防止内存泄漏

这个完整的哈希表实现展示了如何将前面学到的概念应用到实际代码中，包含了错误处理和内存管理的最佳实践。
