# 第19章 并查集

## 19.1 并查集概念

并查集（Disjoint Set Union，DSU）是一种树型的数据结构，用于处理一些不相交集合的合并及查询问题。它主要支持两种操作：查找（Find）和合并（Union）。并查集常被用来解决连通性问题，比如判断图中两个节点是否连通，或者动态维护若干个不相交集合。

### 版本信息表格

| 项目     | 说明                                           |
| -------- | ---------------------------------------------- |
| 数据结构 | 并查集（Disjoint Set Union）                   |
| 适用语言 | C语言                                          |
| 使用场景 | 连通性判断、动态集合维护、图论算法中的组件识别 |

并查集的核心思想是用一棵树来表示一个集合，树的根节点作为集合的代表元素。通过路径压缩和按秩合并等优化技术，并查集的操作效率可以接近常数时间。

### 实例方法表格

| 功能名称   | 实例调用方法       | 具体功能与注意事项                                |
| ---------- | ------------------ | ------------------------------------------------- |
| 初始化     | `init(n)`          | 初始化n个元素的并查集，每个元素初始时自成一个集合 |
| 查找根节点 | `find(x)`          | 查找元素x所在集合的根节点，通常配合路径压缩优化   |
| 合并集合   | `union_sets(a, b)` | 合并包含元素a和b的两个集合                        |

### 使用示例

```c
#include <stdio.h>
#include <stdlib.h>

// 定义并查集结构
typedef struct {
    int *parent;  // parent[i] 表示元素i的父节点
    int *rank;    // rank[i] 表示以i为根的树的高度（用于按秩合并）
    int n;        // 元素总数
} DisjointSet;

// 初始化并查集
// 输入: 元素数量n
// 输出: 初始化后的并查集指针
DisjointSet* init(int n) {
    DisjointSet *ds = (DisjointSet*)malloc(sizeof(DisjointSet));
    if (!ds) {
        fprintf(stderr, "内存分配失败\n");
        return NULL;
    }
    
    ds->n = n;
    ds->parent = (int*)malloc(n * sizeof(int));
    ds->rank = (int*)malloc(n * sizeof(int));
    
    if (!ds->parent || !ds->rank) {
        fprintf(stderr, "内存分配失败\n");
        free(ds);
        return NULL;
    }
    
    // 初始化每个元素的父节点为自己，秩为0
    for (int i = 0; i < n; i++) {
        ds->parent[i] = i;
        ds->rank[i] = 0;
    }
    
    return ds;
}

// 查找根节点（带路径压缩）
// 输入: 并查集指针ds，元素x
// 输出: x所在集合的根节点
int find(DisjointSet *ds, int x) {
    if (x < 0 || x >= ds->n) {
        fprintf(stderr, "元素索引越界\n");
        return -1;
    }
    
    // 路径压缩：将路径上的所有节点直接连接到根节点
    if (ds->parent[x] != x) {
        ds->parent[x] = find(ds, ds->parent[x]);
    }
    return ds->parent[x];
}

// 合并两个集合（按秩合并）
// 输入: 并查集指针ds，要合并的两个元素a和b
// 输出: 无返回值，但会修改并查集结构
void union_sets(DisjointSet *ds, int a, int b) {
    if (a < 0 || a >= ds->n || b < 0 || b >= ds->n) {
        fprintf(stderr, "元素索引越界\n");
        return;
    }
    
    int rootA = find(ds, a);
    int rootB = find(ds, b);
    
    // 如果已经在同一个集合中，无需合并
    if (rootA == rootB) {
        return;
    }
    
    // 按秩合并：将秩小的树连接到秩大的树下面
    if (ds->rank[rootA] < ds->rank[rootB]) {
        ds->parent[rootA] = rootB;
    } else if (ds->rank[rootA] > ds->rank[rootB]) {
        ds->parent[rootB] = rootA;
    } else {
        // 秩相同时，任选一个作为根，并增加其秩
        ds->parent[rootB] = rootA;
        ds->rank[rootA]++;
    }
}

// 释放并查集内存
// 输入: 并查集指针ds
// 输出: 无返回值
void destroy(DisjointSet *ds) {
    if (ds) {
        free(ds->parent);
        free(ds->rank);
        free(ds);
    }
}

// 主函数示例
int main() {
    // 创建包含5个元素的并查集
    DisjointSet *ds = init(5);
    if (!ds) {
        return 1;
    }
    
    // 合并一些元素
    union_sets(ds, 0, 1);  // 合并0和1
    union_sets(ds, 2, 3);  // 合并2和3
    union_sets(ds, 1, 2);  // 合并1和2（间接合并0,1,2,3）
    
    // 检查连通性
    printf("0和3是否连通: %s\n", 
           find(ds, 0) == find(ds, 3) ? "是" : "否");
    printf("0和4是否连通: %s\n", 
           find(ds, 0) == find(ds, 4) ? "是" : "否");
    
    // 释放内存
    destroy(ds);
    return 0;
}
```

### 注意事项

- **初始化范围**：并查集通常使用0到n-1的整数作为元素标识，确保访问时不会越界
- **内存管理**：C语言中需要手动管理内存，记得在使用完毕后释放分配的内存
- **错误处理**：在实际应用中应该添加适当的错误检查，如空指针检查和边界检查
- **性能特点**：经过路径压缩和按秩合并优化后，并查集的单次操作平均时间复杂度接近O(α(n))，其中α是反阿克曼函数，增长极其缓慢

并查集是一种非常实用的数据结构，特别适合处理动态连通性问题。它的实现简洁而高效，是算法竞赛和实际工程中的常用工具。
