## 4.1 顺序结构

### 顺序结构概念

顺序结构是程序中最基本的执行方式，就像我们日常生活中按部就班地做事一样。程序按照代码书写的先后顺序，从上到下依次执行每一条语句，没有跳转，没有分支，规规矩矩，井井有条。

### 顺序执行示例

```c
#include <stdio.h>  // 包含标准输入输出头文件

int main() {  // 主函数入口
    int a = 10;  // 声明整型变量a并初始化为10
    int b = 20;  // 声明整型变量b并初始化为20
    int sum;  // 声明整型变量sum用于存储结果
    
    sum = a + b;  // 计算a和b的和，赋值给sum
    
    printf("a = %d\n", a);  // 输出变量a的值
    printf("b = %d\n", b);  // 输出变量b的值
    printf("sum = %d\n", sum);  // 输出计算结果
    
    return 0;  // 程序正常结束，返回0
}
```

**输出结果：**

```
a = 10
b = 20
sum = 30
```

### 顺序结构特点

1. **执行顺序固定**：严格按照代码书写顺序执行
2. **无条件执行**：每条语句都会被执行，没有选择
3. **简单直观**：逻辑清晰，易于理解和调试
4. **基础结构**：是所有复杂程序的基础

### 注意事项

- 顺序结构虽然简单，但要注意语句的执行顺序对结果的影响
- 变量必须先声明后使用，否则会导致编译错误
- 语句之间用分号隔开，不要遗漏
- 顺序结构通常与其他控制结构结合使用

小结

顺序结构是程序执行的基础，它按照代码书写的顺序依次执行每条语句。虽然简单，但却是构建复杂程序的基石。理解顺序结构有助于我们更好地掌握后续的分支和循环结构。

## 4.2 选择结构（if、if-else、switch）

### 选择结构概述

选择结构让程序有了"思考"的能力，可以根据不同的条件做出不同的选择，就像人生中的十字路口，需要根据情况选择不同的道路。

### if语句

#### 语法格式

```c
if (条件表达式) {
    // 条件为真时执行的代码
}
```

#### 单分支if示例

```c
#include <stdio.h>

int main() {
    int score = 85;  // 定义学生成绩变量
    
    if (score >= 60) {  // 判断成绩是否及格
        printf("恭喜你，考试及格了！\n");  // 条件成立时执行
    }
    
    printf("程序执行完毕。\n");  // 无论条件是否成立都会执行
    
    return 0;
}
```

### if-else语句

#### 语法格式

```c
if (条件表达式) {
    // 条件为真时执行的代码
} else {
    // 条件为假时执行的代码
}
```

#### 双分支示例

```c
#include <stdio.h>

int main() {
    int age = 18;  // 定义年龄变量
    
    if (age >= 18) {  // 判断是否成年
        printf("你已经成年，可以考驾照了！\n");  // 成年执行
    } else {
        printf("你还未成年，需要继续努力学习！\n");  // 未成年执行
    }
    
    return 0;
}
```

### if-else if-else语句

#### 语法格式

```c
if (条件1) {
    // 条件1为真时执行
} else if (条件2) {
    // 条件2为真时执行
} else if (条件3) {
    // 条件3为真时执行
} else {
    // 所有条件都不成立时执行
}
```

#### 多分支示例

```c
#include <stdio.h>

int main() {
    int score = 85;  // 学生成绩
    
    if (score >= 90) {  // 优秀
        printf("成绩等级：优秀 (A)\n");
    } else if (score >= 80) {  // 良好
        printf("成绩等级：良好 (B)\n");
    } else if (score >= 70) {  // 中等
        printf("成绩等级：中等 (C)\n");
    } else if (score >= 60) {  // 及格
        printf("成绩等级：及格 (D)\n");
    } else {  // 不及格
        printf("成绩等级：不及格 (E)\n");
    }
    
    return 0;
}
```

### switch语句

#### 语法格式

```c
switch (表达式) {
    case 常量1:
        // 代码块1
        break;
    case 常量2:
        // 代码块2
        break;
    default:
        // 默认代码块
}
```

#### switch示例

```c
#include <stdio.h>

int main() {
    int day = 3;  // 星期几，1-7
    
    switch (day) {  // 根据day的值进行选择
        case 1:
            printf("今天是星期一，新的一周开始了！\n");
            break;  // 跳出switch语句
        case 2:
            printf("今天是星期二，继续加油！\n");
            break;
        case 3:
            printf("今天是星期三，已经过了一半！\n");
            break;
        case 4:
            printf("今天是星期四，快到周末了！\n");
            break;
        case 5:
            printf("今天是星期五，明天就周末了！\n");
            break;
        case 6:
            printf("今天是星期六，好好休息！\n");
            break;
        case 7:
            printf("今天是星期日，准备迎接新的一周！\n");
            break;
        default:
            printf("输入的日期不正确！\n");  // 默认情况
    }
    
    return 0;
}
```

### 实例方法表格

| 功能名称   | 调用方法         | 具体功能与注意事项                          |
| ---------- | ---------------- | ------------------------------------------- |
| 单分支判断 | if(条件){}       | 条件为真时执行，条件可以是任何表达式        |
| 双分支判断 | if(条件){}else{} | 条件为真执行if，为假执行else                |
| 多分支判断 | if-else if-else  | 按顺序判断，找到第一个为真的条件执行        |
| 多值选择   | switch(表达式){} | 表达式必须是整型或字符型，不能是浮点型      |
| 防止穿透   | break语句        | case后必须加break，否则会继续执行下一个case |

### 嵌套if示例

```c
#include <stdio.h>

int main() {
    int age = 25;  // 年龄
    char gender = 'M';  // 性别，M表示男，F表示女
    
    if (age >= 18) {  // 外层if：判断是否成年
        if (gender == 'M') {  // 内层if：判断性别
            printf("成年男性\n");
        } else {
            printf("成年女性\n");
        }
    } else {
        if (gender == 'M') {
            printf("未成年男性\n");
        } else {
            printf("未成年女性\n");
        }
    }
    
    return 0;
}
```

### 注意事项

1. **if语句注意事项**：
   - 条件表达式可以是任何类型，非零即为真
   - 即使只有一条语句，也建议使用花括号，提高可读性
   - 注意运算符优先级，必要时使用括号
2. **switch语句注意事项**：
   - case后面必须是常量表达式，不能是变量
   - 不要忘记break，否则会发生"穿透"现象
   - default不是必需的，但建议加上以处理意外情况
   - 表达式类型限制：只能是整型、字符型或枚举类型
3. **通用注意事项**：
   - 避免过深的嵌套，影响代码可读性
   - 合理使用注释，说明判断逻辑
   - 注意边界条件的处理

小结

选择结构是程序逻辑控制的核心，通过if、if-else、switch等语句，程序可以根据不同条件做出不同选择。掌握选择结构对于编写灵活、智能的程序至关重要。

## 4.3 循环结构（for、while、do-while）

### 循环结构概述

循环结构让程序具备了"重复劳动"的能力，可以自动执行重复的任务，大大提高了编程效率。就像工厂里的流水线，可以不知疲倦地重复同样的工作。

### for循环

#### 语法格式

```c
for (初始化; 条件判断; 迭代更新) {
    // 循环体
}
```

#### for循环示例

```c
#include <stdio.h>

int main() {
    // 计算1到10的和
    int sum = 0;  // 初始化求和变量
    
    for (int i = 1; i <= 10; i++) {  // i从1到10循环
        sum += i;  // 累加到sum
        printf("当前累加到 %d，总和为 %d\n", i, sum);  // 输出当前状态
    }
    
    printf("1到10的总和是：%d\n", sum);  // 输出最终结果
    
    return 0;
}
```

#### 嵌套for循环示例

```c
#include <stdio.h>

int main() {
    // 打印九九乘法表
    for (int i = 1; i <= 9; i++) {  // 外层循环：行数
        for (int j = 1; j <= i; j++) {  // 内层循环：列数
            printf("%d×%d=%2d  ", j, i, i * j);  // 打印乘法表达式
        }
        printf("\n");  // 换行
    }
    
    return 0;
}
```

### while循环

#### 语法格式

```c
while (条件表达式) {
    // 循环体
}
```

#### while循环示例

```c
#include <stdio.h>

int main() {
    int count = 1;  // 初始化计数器
    
    while (count <= 5) {  // 当count小于等于5时循环
        printf("第 %d 次循环\n", count);  // 输出循环次数
        count++;  // 计数器自增
    }
    
    printf("循环结束，总共执行了 %d 次\n", count - 1);  // 输出总次数
    
    return 0;
}
```

#### while循环应用：猜数字游戏

```c
#include <stdio.h>
#include <stdlib.h>
#include <time.h>

int main() {
    srand(time(NULL));  // 初始化随机数种子
    int secret = rand() % 100 + 1;  // 生成1-100的随机数
    int guess;
    int attempts = 0;  // 记录尝试次数
    
    printf("欢迎来到猜数字游戏！我已经想好了一个1-100之间的数字。\n");
    
    while (1) {  // 无限循环，直到猜中为止
        printf("请输入你的猜测：");
        
        if (scanf("%d", &guess) != 1) {  // 检查输入是否有效
            printf("输入无效，请输入一个整数！\n");
            while (getchar() != '\n');  // 清空输入缓冲区
            continue;  // 跳过本次循环
        }
        
        attempts++;  // 尝试次数加1
        
        if (guess < secret) {
            printf("太小了！再试一次。\n");
        } else if (guess > secret) {
            printf("太大了！再试一次。\n");
        } else {
            printf("恭喜你！猜中了！答案就是 %d\n", secret);
            printf("你总共尝试了 %d 次\n", attempts);
            break;  // 猜中后退出循环
        }
    }
    
    return 0;
}
```

### do-while循环

#### 语法格式

```c
do {
    // 循环体
} while (条件表达式);
```

#### do-while循环示例

```c
#include <stdio.h>

int main() {
    int number;
    
    // do-while保证至少执行一次
    do {
        printf("请输入一个正整数：");
        scanf("%d", &number);
        
        if (number <= 0) {
            printf("输入无效！请重新输入。\n");
        }
    } while (number <= 0);  // 条件为真时继续循环
    
    printf("你输入的正整数是：%d\n", number);
    
    return 0;
}
```

### 三种循环对比

| 循环类型     | 特点                           | 适用场景                   |
| ------------ | ------------------------------ | -------------------------- |
| for循环      | 结构清晰，适合已知循环次数     | 遍历数组、固定次数循环     |
| while循环    | 先判断后执行，可能一次都不执行 | 条件满足时循环，次数不确定 |
| do-while循环 | 先执行后判断，至少执行一次     | 需要至少执行一次的场景     |

### 循环控制示例：素数判断

```c
#include <stdio.h>
#include <stdbool.h>

int main() {
    int num;
    
    printf("请输入一个整数：");
    scanf("%d", &num);
    
    if (num <= 1) {
        printf("%d 不是素数。\n", num);
        return 0;
    }
    
    bool isPrime = true;  // 假设是素数
    
    // 只需要检查到sqrt(num)即可
    for (int i = 2; i * i <= num; i++) {
        if (num % i == 0) {  // 如果能被i整除
            isPrime = false;  // 不是素数
            break;  // 提前退出循环
        }
    }
    
    if (isPrime) {
        printf("%d 是素数。\n", num);
    } else {
        printf("%d 不是素数。\n", num);
    }
    
    return 0;
}
```

### 实例方法表格

| 功能名称     | 调用方法                | 具体功能与注意事项                   |
| ------------ | ----------------------- | ------------------------------------ |
| 固定次数循环 | for(初始化;条件;更新){} | 适合已知循环次数的场景               |
| 条件循环     | while(条件){}           | 条件为真时循环，可能一次都不执行     |
| 至少执行一次 | do{}while(条件);        | 先执行后判断，保证至少执行一次       |
| 提前退出循环 | break语句               | 立即退出当前循环                     |
| 跳过本次循环 | continue语句            | 跳过本次循环剩余代码，进入下一次循环 |

### 注意事项

1. **避免死循环**：
   - 确保循环条件最终会变为假
   - 检查循环变量是否正确更新
   - 使用调试工具检查循环执行情况
2. **循环效率**：
   - 避免在循环条件中进行复杂计算
   - 减少不必要的循环嵌套
   - 合理使用break提前退出
3. **代码可读性**：
   - 选择合适的循环类型
   - 添加必要的注释
   - 保持循环体简洁
4. **边界条件**：
   - 注意循环的起始和结束条件
   - 处理空循环的情况
   - 检查循环变量的溢出问题

小结

循环结构是程序自动化的关键，通过for、while、do-while三种循环，可以高效地处理重复任务。掌握循环的使用技巧，能够大大提升编程效率和代码质量。

## 4.4 跳转语句（break、continue、goto）

### 跳转语句概述

跳转语句就像程序中的"传送门"，可以让我们在程序的不同位置之间快速移动，实现更灵活的流程控制。不过，使用时要谨慎，避免造成代码混乱。

### break语句

#### 功能说明

break语句用于立即退出当前的循环或switch语句，就像紧急刹车一样，让程序立即停止当前的执行流程。

#### break在循环中的应用

```c
#include <stdio.h>

int main() {
    // 查找第一个能被7整除的数
    for (int i = 1; i <= 100; i++) {
        if (i % 7 == 0) {  // 如果能被7整除
            printf("找到第一个能被7整除的数：%d\n", i);
            break;  // 立即退出循环
        }
    }
    
    // 遍历数组查找特定元素
    int numbers[] = {10, 20, 30, 40, 50, 60, 70};
    int target = 40;
    int found = 0;
    
    for (int i = 0; i < 7; i++) {
        if (numbers[i] == target) {
            printf("在索引 %d 处找到目标值 %d\n", i, target);
            found = 1;
            break;  // 找到后立即退出
        }
    }
    
    if (!found) {
        printf("未找到目标值 %d\n", target);
    }
    
    return 0;
}
```

#### break在switch中的应用

```c
#include <stdio.h>

int main() {
    char grade = 'B';
    
    switch (grade) {
        case 'A':
            printf("优秀！\n");
            break;  // 防止穿透到下一个case
        case 'B':
            printf("良好！\n");
            break;
        case 'C':
            printf("中等！\n");
            break;
        case 'D':
            printf("及格！\n");
            break;
        default:
            printf("不及格！\n");
    }
    
    return 0;
}
```

### continue语句

#### 功能说明

continue语句用于跳过当前循环的剩余部分，直接进入下一次循环迭代，就像说"这次不算，再来一次"。

#### continue应用示例

```c
#include <stdio.h>

int main() {
    // 打印1-20之间的奇数
    printf("1-20之间的奇数：\n");
    for (int i = 1; i <= 20; i++) {
        if (i % 2 == 0) {  // 如果是偶数
            continue;  // 跳过本次循环
        }
        printf("%d ", i);  // 只打印奇数
    }
    printf("\n");
    
    // 计算数组中正数的和
    int numbers[] = {-5, 10, -3, 20, 0, 15, -8, 25};
    int sum = 0;
    
    for (int i = 0; i < 8; i++) {
        if (numbers[i] <= 0) {  // 如果不是正数
            continue;  // 跳过
        }
        sum += numbers[i];  // 累加正数
    }
    
    printf("正数的和为：%d\n", sum);
    
    return 0;
}
```

### goto语句

#### 功能说明

goto语句是C语言中最强大的跳转语句，可以直接跳转到程序中的任何标记位置。但由于容易造成代码混乱，通常不推荐使用。

#### goto语法

```c
goto 标签名;
...
标签名: 语句;
```

#### goto应用示例

```c
#include <stdio.h>

int main() {
    int choice;
    
    menu:  // 菜单标签
    printf("\n=== 简易计算器 ===\n");
    printf("1. 加法\n");
    printf("2. 减法\n");
    printf("3. 乘法\n");
    printf("4. 除法\n");
    printf("0. 退出\n");
    printf("请选择操作：");
    
    scanf("%d", &choice);
    
    if (choice == 0) {
        printf("感谢使用，再见！\n");
        goto end;  // 跳转到结束标签
    }
    
    double num1, num2, result;
    printf("请输入两个数字：");
    scanf("%lf %lf", &num1, &num2);
    
    switch (choice) {
        case 1:
            result = num1 + num2;
            printf("%.2f + %.2f = %.2f\n", num1, num2, result);
            break;
        case 2:
            result = num1 - num2;
            printf("%.2f - %.2f = %.2f\n", num1, num2, result);
            break;
        case 3:
            result = num1 * num2;
            printf("%.2f × %.2f = %.2f\n", num1, num2, result);
            break;
        case 4:
            if (num2 == 0) {
                printf("错误：除数不能为0！\n");
                goto menu;  // 重新显示菜单
            }
            result = num1 / num2;
            printf("%.2f ÷ %.2f = %.2f\n", num1, num2, result);
            break;
        default:
            printf("无效的选择！\n");
            goto menu;  // 重新显示菜单
    }
    
    goto menu;  // 继续循环
    
    end:  // 结束标签
    return 0;
}
```

### 多层循环中的break和continue

```c
#include <stdio.h>

int main() {
    // 使用标志变量控制多层循环退出
    int found = 0;
    
    for (int i = 1; i <= 5 && !found; i++) {
        for (int j = 1; j <= 5 && !found; j++) {
            printf("i=%d, j=%d\n", i, j);
            
            if (i == 3 && j == 3) {
                printf("找到目标位置！\n");
                found = 1;  // 设置标志
                break;  // 退出内层循环
            }
        }
    }
    
    // 使用goto退出多层循环（不推荐但可行）
    printf("\n使用goto退出多层循环：\n");
    
    for (int i = 1; i <= 5; i++) {
        for (int j = 1; j <= 5; j++) {
            printf("i=%d, j=%d\n", i, j);
            
            if (i == 3 && j == 3) {
                printf("找到目标，使用goto退出！\n");
                goto exit_loops;  // 直接跳转到标签
            }
        }
    }
    
    exit_loops:  // 退出标签
    printf("已退出所有循环。\n");
    
    return 0;
}
```

### 实例方法表格

| 功能名称        | 调用方法     | 具体功能与注意事项                   |
| --------------- | ------------ | ------------------------------------ |
| 退出循环/switch | break;       | 立即退出当前最内层的循环或switch     |
| 跳过本次循环    | continue;    | 跳过当前循环剩余代码，进入下一次迭代 |
| 无条件跳转      | goto 标签名; | 跳转到指定标签位置，慎用             |
| 标签定义        | 标签名:      | 定义跳转目标位置                     |

### 注意事项

1. **break语句**：
   - 只能退出最内层的循环或switch
   - 在多层循环中需要配合标志变量使用
   - 在switch中防止"穿透"现象
2. **continue语句**：
   - 只影响当前循环的本次迭代
   - 在for循环中会先执行迭代表达式
   - 在while和do-while中直接判断条件
3. **goto语句**：
   - 尽量避免使用，容易造成代码混乱
   - 只在特定场景下使用（如错误处理、多层循环退出）
   - 标签名必须在当前函数内定义
   - 不要跨函数跳转
4. **最佳实践**：
   - 优先使用break和continue进行循环控制
   - 避免过度使用goto，保持代码结构清晰
   - 使用有意义的标签名
   - 在团队开发中遵循统一的编码规范

小结

跳转语句提供了灵活的程序流程控制能力。break和continue是循环控制的好帮手，而goto虽然强大但要慎用。合理使用跳转语句可以让代码更加简洁高效。

## 4.5 条件运算符（三目运算符）

### 条件运算符概述

条件运算符（也称为三目运算符）是C语言中唯一的三元运算符，它提供了一种简洁的方式来表达简单的条件判断，就像编程中的"速记法"。

### 语法格式

```c
条件表达式 ? 表达式1 : 表达式2
```

**执行逻辑**：如果条件表达式为真（非零），则计算并返回表达式1的值；否则计算并返回表达式2的值。

### 基本使用示例

```c
#include <stdio.h>

int main() {
    int a = 10, b = 20;
    
    // 使用条件运算符找出较大值
    int max = (a > b) ? a : b;  // 如果a>b，max=a，否则max=b
    printf("较大值是：%d\n", max);
    
    // 使用条件运算符找出较小值
    int min = (a < b) ? a : b;  // 如果a<b，min=a，否则min=b
    printf("较小值是：%d\n", min);
    
    // 判断奇偶数
    int num = 15;
    char *result = (num % 2 == 0) ? "偶数" : "奇数";
    printf("%d 是 %s\n", num, result);
    
    return 0;
}
```

### 条件运算符与if-else对比

```c
#include <stdio.h>

int main() {
    int score = 85;
    
    // 使用if-else
    char grade1;
    if (score >= 60) {
        grade1 = 'P';  // 及格
    } else {
        grade1 = 'F';  // 不及格
    }
    printf("if-else方式：成绩等级 %c\n", grade1);
    
    // 使用条件运算符
    char grade2 = (score >= 60) ? 'P' : 'F';
    printf("条件运算符方式：成绩等级 %c\n", grade2);
    
    // 字符串版本
    char *status = (score >= 60) ? "及格" : "不及格";
    printf("成绩状态：%s\n", status);
    
    return 0;
}
```

### 嵌套条件运算符

```c
#include <stdio.h>

int main() {
    int score = 85;
    
    // 嵌套使用：判断成绩等级
    char grade = (score >= 90) ? 'A' :
                 (score >= 80) ? 'B' :
                 (score >= 70) ? 'C' :
                 (score >= 60) ? 'D' : 'E';
    
    printf("成绩：%d，等级：%c\n", score, grade);
    
    // 可读性更好的写法
    char grade2 = (score >= 90) ? 'A'
                : (score >= 80) ? 'B'
                : (score >= 70) ? 'C'
                : (score >= 60) ? 'D'
                : 'E';
    
    printf("成绩：%d，等级（格式化）：%c\n", score, grade2);
    
    return 0;
}
```

### 实际应用场景

```c
#include <stdio.h>
#include <string.h>

int main() {
    // 1. 绝对值计算
    int num = -15;
    int abs_value = (num < 0) ? -num : num;
    printf("%d 的绝对值是 %d\n", num, abs_value);
    
    // 2. 数组边界检查
    int index = 10;
    int max_index = 9;
    int safe_index = (index > max_index) ? max_index : index;
    printf("安全索引：%d\n", safe_index);
    
    // 3. 字符串空值处理
    char *name = NULL;
    char *display_name = (name != NULL) ? name : "匿名用户";
    printf("显示名称：%s\n", display_name);
    
    // 4. 最大值/最小值函数
    int x = 5, y = 12, z = 8;
    int max = (x > y) ? ((x > z) ? x : z) : ((y > z) ? y : z);
    printf("三个数中的最大值：%d\n", max);
    
    // 5. 符号判断
    int value = -7;
    char sign = (value > 0) ? '+' : (value < 0) ? '-' : '0';
    printf("%d 的符号是：%c\n", value, sign);
    
    return 0;
}
```

### 条件运算符在函数参数中的应用

```c
#include <stdio.h>

// 计算两个数的较大值
int max(int a, int b) {
    return (a > b) ? a : b;
}

// 计算两个数的较小值
int min(int a, int b) {
    return (a < b) ? a : b;
}

int main() {
    int a = 15, b = 25;
    
    printf("max(%d, %d) = %d\n", a, b, max(a, b));
    printf("min(%d, %d) = %d\n", a, b, min(a, b));
    
    // 在printf中直接使用
    printf("较大值是：%d\n", (a > b) ? a : b);
    
    return 0;
}
```

### 实例方法表格

| 功能名称     | 使用方法                | 具体功能与注意事项         |
| ------------ | ----------------------- | -------------------------- |
| 简单条件判断 | 条件?值1:值2            | 替代简单的if-else，返回值  |
| 嵌套条件判断 | 条件1?值1:条件2?值2:值3 | 多层条件判断，从右向左结合 |
| 绝对值计算   | (x<0)?-x:x              | 快速计算绝对值             |
| 边界检查     | (index>max)?max:index   | 防止数组越界               |
| 空值处理     | (ptr)?ptr:"default"     | 处理可能为空的指针         |

### 注意事项

1. **类型一致性**：
   - 表达式1和表达式2的类型应该兼容
   - 如果类型不同，会进行隐式类型转换
   - 返回值的类型由两个表达式中较高级的类型决定
2. **可读性**：
   - 只用于简单的条件判断
   - 避免过深的嵌套，影响代码可读性
   - 复杂逻辑还是使用if-else更清晰
3. **副作用**：
   - 只有被选中的表达式会被求值
   - 未被选中的表达式不会执行，不会有副作用
4. **优先级**：
   - 条件运算符的优先级较低
   - 在复杂表达式中建议使用括号明确优先级
   - 赋值运算符的优先级低于条件运算符
5. **最佳实践**：
   - 用于简单的二选一场景
   - 保持表达式简洁
   - 适当添加括号提高可读性
   - 在团队开发中遵循统一的编码风格
