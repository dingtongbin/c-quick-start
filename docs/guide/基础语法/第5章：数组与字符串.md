## 5.1 一维数组的定义与使用

### 一维数组实例方法

| 功能名称     | 实例调用方法                 | 具体功能与注意事项                         |
| ------------ | ---------------------------- | ------------------------------------------ |
| 数组定义     | `int arr[10];`               | 定义包含10个整数的数组，未初始化时值不确定 |
| 数组初始化   | `int arr[5] = {1,2,3,4,5};`  | 定义并初始化数组，可省略数组大小           |
| 访问元素     | `arr[0]`                     | 通过下标访问数组元素，下标从0开始          |
| 遍历数组     | `for(int i=0; i<5; i++)`     | 使用循环遍历数组所有元素                   |
| 计算数组长度 | `sizeof(arr)/sizeof(arr[0])` | 计算数组元素个数，仅在定义处有效           |

### 使用示例

```c
#include <stdio.h>  // 引入标准输入输出头文件，用于printf和scanf函数

int main() {
    // 定义一个包含5个整数的一维数组，未初始化
    int numbers[5];
    
    // 通过下标为数组元素赋值，下标从0开始
    numbers[0] = 10;
    numbers[1] = 20;
    numbers[2] = 30;
    numbers[3] = 40;
    numbers[4] = 50;
    
    // 使用for循环遍历数组，i从0到4
    printf("数组元素:\n");
    for (int i = 0; i < 5; i++) {
        // 输出每个数组元素的值
        printf("numbers[%d] = %d\n", i, numbers[i]);
    }
    
    // 定义并初始化数组，编译器自动计算数组大小
    int scores[] = {85, 92, 78, 95, 88, 90};
    
    // 计算数组长度：总字节数除以单个元素字节数
    int length = sizeof(scores) / sizeof(scores[0]);
    
    // 输出数组长度
    printf("\n数组scores的长度: %d\n", length);
    
    // 计算数组元素总和
    int sum = 0;
    for (int i = 0; i < length; i++) {
        // 累加每个元素到sum
        sum += scores[i];
    }
    
    // 计算并输出平均值
    float average = (float)sum / length;
    printf("数组scores的平均值: %.2f\n", average);
    
    return 0;  // 程序正常结束，返回0
}
```

### 注意事项

1. **数组下标从0开始**：C语言数组的第一个元素下标是0，最后一个元素下标是n-1
2. **越界访问危险**：访问超出数组范围的元素会导致未定义行为，可能程序崩溃
3. **数组大小必须是常量**：在C99之前，数组大小必须是编译时常量
4. **初始化不完全**：如果初始化列表元素少于数组大小，剩余元素自动初始化为0
5. **数组名是地址**：数组名代表数组首元素的地址，不能被修改

### 小结

一维数组是C语言中最基本的数据结构，用于存储相同类型的数据集合。通过下标可以快速访问和修改数组元素，配合循环可以高效地处理批量数据。掌握数组的定义、初始化和遍历是学习C语言的重要基础。

## 5.2 二维数组与多维数组

### 二维数组实例方法

| 功能名称       | 实例调用方法                         | 具体功能与注意事项                  |
| -------------- | ------------------------------------ | ----------------------------------- |
| 二维数组定义   | `int matrix[3][4];`                  | 定义3行4列的二维数组                |
| 二维数组初始化 | `int arr[2][3] = {{1,2,3},{4,5,6}};` | 按行初始化二维数组                  |
| 访问元素       | `matrix[1][2]`                       | 访问第2行第3列的元素（下标从0开始） |
| 遍历二维数组   | 嵌套for循环                          | 外层循环行，内层循环列              |
| 不规则二维数组 | 指针数组实现                         | 每行可以有不同的列数                |

### 使用示例

```c
#include <stdio.h>  // 引入标准输入输出头文件

int main() {
    // 定义并初始化一个3行4列的二维数组
    int matrix[3][4] = {
        {1, 2, 3, 4},    // 第1行
        {5, 6, 7, 8},    // 第2行
        {9, 10, 11, 12}  // 第3行
    };
    
    // 输出二维数组的行数和列数
    printf("二维数组信息:\n");
    printf("行数: %d\n", 3);
    printf("列数: %d\n", 4);
    
    // 使用嵌套for循环遍历二维数组
    printf("\n二维数组内容:\n");
    for (int i = 0; i < 3; i++) {  // 外层循环控制行
        for (int j = 0; j < 4; j++) {  // 内层循环控制列
            // 输出当前元素，宽度为4个字符，右对齐
            printf("%4d", matrix[i][j]);
        }
        // 每行结束后换行
        printf("\n");
    }
    
    // 计算每行的和
    printf("\n每行的和:\n");
    for (int i = 0; i < 3; i++) {
        int rowSum = 0;  // 初始化行和为0
        for (int j = 0; j < 4; j++) {
            // 累加当前行的所有元素
            rowSum += matrix[i][j];
        }
        // 输出第i行的和
        printf("第 %d 行的和: %d\n", i + 1, rowSum);
    }
    
    // 计算每列的和
    printf("\n每列的和:\n");
    for (int j = 0; j < 4; j++) {
        int colSum = 0;  // 初始化列和为0
        for (int i = 0; i < 3; i++) {
            // 累加当前列的所有元素
            colSum += matrix[i][j];
        }
        // 输出第j列的和
        printf("第 %d 列的和: %d\n", j + 1, colSum);
    }
    
    // 定义一个不规则的二维数组（使用指针数组）
    int *irregular[3];  // 定义包含3个指针的数组
    
    // 为每行动态分配不同大小的内存
    irregular[0] = (int *)malloc(2 * sizeof(int));  // 第1行2个元素
    irregular[1] = (int *)malloc(3 * sizeof(int));  // 第2行3个元素
    irregular[2] = (int *)malloc(4 * sizeof(int));  // 第3行4个元素
    
    // 初始化不规则数组
    irregular[0][0] = 1; irregular[0][1] = 2;
    irregular[1][0] = 3; irregular[1][1] = 4; irregular[1][2] = 5;
    irregular[2][0] = 6; irregular[2][1] = 7; irregular[2][2] = 8; irregular[2][3] = 9;
    
    // 输出不规则数组
    printf("\n不规则二维数组:\n");
    printf("第1行(2个元素): %d %d\n", irregular[0][0], irregular[0][1]);
    printf("第2行(3个元素): %d %d %d\n", irregular[1][0], irregular[1][1], irregular[1][2]);
    printf("第3行(4个元素): %d %d %d %d\n", irregular[2][0], irregular[2][1], irregular[2][2], irregular[2][3]);
    
    // 释放动态分配的内存
    free(irregular[0]);
    free(irregular[1]);
    free(irregular[2]);
    
    return 0;  // 程序正常结束
}
```

### 注意事项

1. **内存布局**：二维数组在内存中是按行连续存储的
2. **初始化规则**：可以只初始化部分元素，未初始化的自动为0
3. **数组大小限制**：过大的数组可能导致栈溢出，应使用动态内存分配
4. **访问效率**：按行访问比按列访问效率更高（缓存友好）
5. **多维数组**：可以定义三维及以上数组，但使用频率较低

### 小结

二维数组是数组的数组，常用于表示矩阵和表格数据。通过嵌套循环可以高效地遍历和操作二维数组。不规则二维数组提供了更大的灵活性，适用于每行数据量不同的场景。理解二维数组的内存布局对于优化程序性能很重要。

## 5.3 字符数组与字符串处理

### 字符数组实例方法

| 功能名称     | 实例调用方法             | 具体功能与注意事项                 |
| ------------ | ------------------------ | ---------------------------------- |
| 字符数组定义 | `char str[20];`          | 定义可以存储19个字符加结束符的数组 |
| 字符串初始化 | `char str[] = "Hello";`  | 自动添加'\0'结束符，数组大小为6    |
| 单字符赋值   | `str[0] = 'A';`          | 为单个字符位置赋值                 |
| 字符串输入   | `scanf("%s", str);`      | 读取字符串，遇到空格停止           |
| 安全输入     | `fgets(str, 20, stdin);` | 读取整行，包括空格，更安全         |

### 使用示例

```c
#include <stdio.h>  // 引入标准输入输出头文件

int main() {
    // 定义并初始化字符数组，自动添加'\0'结束符
    char greeting[] = "Hello, World!";
    
    // 输出字符串
    printf("问候语: %s\n", greeting);
    
    // 输出字符串长度（不包括结束符）
    printf("字符串长度: %lu\n", sizeof(greeting) - 1);
    
    // 逐个字符输出
    printf("\n逐个字符输出:\n");
    for (int i = 0; greeting[i] != '\0'; i++) {
        // 输出每个字符及其ASCII码
        printf("字符 '%c' 的ASCII码: %d\n", greeting[i], greeting[i]);
    }
    
    // 定义一个可以存储用户输入的字符数组
    char name[50];
    
    // 提示用户输入
    printf("\n请输入您的名字: ");
    
    // 使用fgets安全读取整行输入，包括空格
    fgets(name, sizeof(name), stdin);
    
    // 去除输入末尾的换行符（如果存在）
    int len = 0;
    while (name[len] != '\0') {
        if (name[len] == '\n') {
            name[len] = '\0';  // 将换行符替换为结束符
            break;
        }
        len++;
    }
    
    // 输出用户输入的名字
    printf("您好, %s!\n", name);
    
    // 定义两个字符数组用于演示字符串拼接
    char firstName[20] = "张";
    char lastName[20] = "三";
    char fullName[40];
    
    // 手动拼接字符串
    int i = 0;
    // 复制firstName到fullName
    while (firstName[i] != '\0') {
        fullName[i] = firstName[i];
        i++;
    }
    
    // 复制lastName到fullName
    int j = 0;
    while (lastName[j] != '\0') {
        fullName[i] = lastName[j];
        i++;
        j++;
    }
    
    // 添加字符串结束符
    fullName[i] = '\0';
    
    // 输出拼接后的字符串
    printf("\n姓名拼接: %s + %s = %s\n", firstName, lastName, fullName);
    
    // 演示字符数组的修改
    char text[20] = "abcdef";
    printf("\n原始文本: %s\n", text);
    
    // 修改特定位置的字符
    text[0] = 'A';  // 将第一个字符改为大写
    text[3] = 'D';  // 将第四个字符改为大写
    
    printf("修改后文本: %s\n", text);
    
    // 演示字符串比较（手动实现）
    char str1[20] = "apple";
    char str2[20] = "banana";
    
    printf("\n比较字符串: \"%s\" 和 \"%s\"\n", str1, str2);
    
    int k = 0;
    while (str1[k] != '\0' && str2[k] != '\0') {
        if (str1[k] != str2[k]) {
            break;
        }
        k++;
    }
    
    // 根据比较结果输出
    if (str1[k] == '\0' && str2[k] == '\0') {
        printf("两个字符串相等\n");
    } else if (str1[k] == '\0' || str1[k] < str2[k]) {
        printf("\"%s\" 小于 \"%s\"\n", str1, str2);
    } else {
        printf("\"%s\" 大于 \"%s\"\n", str1, str2);
    }
    
    return 0;  // 程序正常结束
}
```

### 注意事项

1. **字符串结束符**：C语言字符串以'\0'（空字符）结尾，必须预留空间
2. **缓冲区溢出**：输入字符串时要确保不会超出数组大小
3. **字符串常量**：字符串字面量存储在只读内存区域，不能修改
4. **空格处理**：scanf读取字符串遇到空格会停止，fgets可以读取整行
5. **内存管理**：使用字符数组时要注意手动管理内存和结束符

### 小结

字符数组是C语言中处理字符串的基础，通过'\0'结束符来标识字符串的结束。理解字符数组的定义、初始化和操作方法对于文本处理至关重要。手动实现字符串操作有助于深入理解字符串的本质，但在实际开发中通常使用标准库函数。

## 5.4 常用字符串函数（strlen、strcpy、strcmp等）

### 字符串函数实例方法

| 功能名称   | 函数原型                                                 | 具体功能与注意事项              |
| ---------- | -------------------------------------------------------- | ------------------------------- |
| 计算长度   | `size_t strlen(const char *s)`                           | 返回字符串长度，不包括'\0'      |
| 复制字符串 | `char *strcpy(char *dest, const char *src)`              | 复制src到dest，包括'\0'         |
| 安全复制   | `char *strncpy(char *dest, const char *src, size_t n)`   | 复制最多n个字符，可能不添加'\0' |
| 连接字符串 | `char *strcat(char *dest, const char *src)`              | 将src追加到dest末尾             |
| 比较字符串 | `int strcmp(const char *s1, const char *s2)`             | 比较两个字符串，返回负/0/正     |
| 查找字符   | `char *strchr(const char *s, int c)`                     | 查找字符c首次出现的位置         |
| 查找子串   | `char *strstr(const char *haystack, const char *needle)` | 查找子串首次出现的位置          |

### 使用示例

```c
#include <stdio.h>   // 引入标准输入输出头文件
#include <string.h>  // 引入字符串处理函数头文件

int main() {
    // 演示strlen函数：计算字符串长度
    char str1[] = "Hello, C Language!";
    size_t length = strlen(str1);  // 计算字符串长度，不包括'\0'
    printf("字符串: \"%s\"\n", str1);
    printf("字符串长度: %zu (不包括结束符)\n", length);
    printf("数组大小: %zu (包括结束符)\n", sizeof(str1));
    
    // 演示strcpy函数：复制字符串
    char source[] = "This is source string";
    char destination[50];
    
    strcpy(destination, source);  // 将source复制到destination
    printf("\n复制后: %s\n", destination);
    
    // 演示strncpy函数：安全复制
    char src[] = "Hello World";
    char dest[10];
    
    // 复制最多9个字符，留1个位置给'\0'
    strncpy(dest, src, 9);
    dest[9] = '\0';  // 手动添加结束符
    printf("\n安全复制(最多9字符): %s\n", dest);
    
    // 演示strcat函数：连接字符串
    char firstName[30] = "John";
    char lastName[] = " Doe";
    
    strcat(firstName, lastName);  // 将lastName追加到firstName末尾
    printf("\n连接后: %s\n", firstName);
    
    // 演示strncat函数：安全连接
    char str2[20] = "Hello";
    char str3[] = " World!";
    
    // 追加最多5个字符，确保不超过数组大小
    strncat(str2, str3, 5);
    printf("安全连接: %s\n", str2);
    
    // 演示strcmp函数：比较字符串
    char str4[] = "apple";
    char str5[] = "banana";
    char str6[] = "apple";
    
    int result1 = strcmp(str4, str5);  // 比较str4和str5
    int result2 = strcmp(str4, str6);  // 比较str4和str6
    
    printf("\n字符串比较:\n");
    printf("\"%s\" vs \"%s\": %d (%s)\n", str4, str5, result1, 
           result1 < 0 ? "小于" : result1 > 0 ? "大于" : "等于");
    printf("\"%s\" vs \"%s\": %d (%s)\n", str4, str6, result2,
           result2 < 0 ? "小于" : result2 > 0 ? "大于" : "等于");
    
    // 演示strncmp函数：比较指定长度
    char str7[] = "Hello World";
    char str8[] = "Hello C";
    
    int result3 = strncmp(str7, str8, 5);  // 比较前5个字符
    printf("\n前5个字符比较: %d (%s)\n", result3,
           result3 == 0 ? "相同" : "不同");
    
    // 演示strchr函数：查找字符
    char str9[] = "Hello, World!";
    char *pos = strchr(str9, 'W');  // 查找字符'W'的位置
    
    if (pos != NULL) {
        // 输出找到的字符及其后续内容
        printf("\n找到字符 'W' 在位置: %ld\n", pos - str9);
        printf("从该位置开始的字符串: %s\n", pos);
    } else {
        printf("\n未找到字符 'W'\n");
    }
    
    // 演示strstr函数：查找子串
    char text[] = "C programming is fun and powerful";
    char *substring = strstr(text, "programming");  // 查找子串
    
    if (substring != NULL) {
        // 输出找到的子串位置和内容
        printf("\n找到子串 \"programming\" 在位置: %ld\n", substring - text);
        printf("从该位置开始的字符串: %s\n", substring);
    } else {
        printf("\n未找到子串\n");
    }
    
    // 演示strtok函数：字符串分割
    char sentence[] = "This is a sample sentence";
    char *token;
    
    printf("\n分割字符串: \"%s\"\n", sentence);
    printf("按空格分割:\n");
    
    // 第一次调用传入字符串，后续传入NULL
    token = strtok(sentence, " ");
    while (token != NULL) {
        printf("  词汇: %s\n", token);
        token = strtok(NULL, " ");  // 继续分割
    }
    
    // 演示memset函数：内存设置
    char buffer[20];
    memset(buffer, '*', 10);  // 将前10个字节设置为'*'
    buffer[10] = '\0';  // 添加结束符
    printf("\nmemset结果: %s\n", buffer);
    
    // 演示memcpy函数：内存复制
    char srcData[] = "1234567890";
    char destData[15];
    
    memcpy(destData, srcData, 5);  // 复制5个字节
    destData[5] = '\0';  // 添加结束符
    printf("memcpy结果: %s\n", destData);
    
    return 0;  // 程序正常结束
}
```

### 注意事项

1. **缓冲区溢出风险**：strcpy、strcat等函数不检查目标缓冲区大小
2. **结束符处理**：strncpy可能不自动添加'\0'，需要手动处理
3. **返回值检查**：查找函数返回NULL表示未找到，必须检查
4. **修改原字符串**：strtok会修改原字符串，使用前需备份
5. **线程安全**：strtok不是线程安全的，多线程环境使用strtok_r

### 小结

标准库提供的字符串函数大大简化了字符串操作。strlen、strcpy、strcat、strcmp等是使用频率最高的函数。理解每个函数的功能、参数和返回值对于正确使用至关重要。在实际开发中，要注意安全性问题，优先使用带长度限制的安全版本函数。

## 5.5 数组的遍历与常见算法

### 数组算法实例方法

| 功能名称 | 算法描述               | 时间复杂度 | 适用场景     |
| -------- | ---------------------- | ---------- | ------------ |
| 线性查找 | 逐个比较查找目标值     | O(n)       | 无序数组     |
| 二分查找 | 折半查找，要求数组有序 | O(log n)   | 有序数组     |
| 冒泡排序 | 相邻元素比较交换       | O(n²)      | 小规模数据   |
| 选择排序 | 每次选择最小元素       | O(n²)      | 小规模数据   |
| 插入排序 | 逐个插入到正确位置     | O(n²)      | 部分有序数据 |
| 数组反转 | 首尾交换               | O(n)       | 数据倒置     |

### 使用示例

```c
#include <stdio.h>  // 引入标准输入输出头文件

// 函数声明：线性查找
int linearSearch(int arr[], int size, int target);

// 函数声明：二分查找（要求数组已排序）
int binarySearch(int arr[], int size, int target);

// 函数声明：冒泡排序
void bubbleSort(int arr[], int size);

// 函数声明：选择排序
void selectionSort(int arr[], int size);

// 函数声明：数组反转
void reverseArray(int arr[], int size);

// 函数声明：打印数组
void printArray(int arr[], int size);

int main() {
    // 定义测试数组
    int numbers[] = {64, 34, 25, 12, 22, 11, 90, 88, 45, 50};
    int size = sizeof(numbers) / sizeof(numbers[0]);
    
    printf("原始数组:\n");

// 线性查找函数实现
int linearSearch(int arr[], int size, int target) {
    // 遍历数组查找目标值
    for (int i = 0; i < size; i++) {
        if (arr[i] == target) {
            return i;  // 找到返回索引
        }
    }
    return -1;  // 未找到返回-1
}

// 二分查找函数实现（要求数组已排序）
int binarySearch(int arr[], int size, int target) {
    int left = 0;           // 左边界
    int right = size - 1;   // 右边界
    
    // 当左边界不超过右边界时继续查找
    while (left <= right) {
        // 计算中间位置，避免溢出
        int mid = left + (right - left) / 2;
        
        if (arr[mid] == target) {
            return mid;  // 找到返回索引
        } else if (arr[mid] < target) {
            left = mid + 1;  // 目标在右半部分
        } else {
            right = mid - 1;  // 目标在左半部分
        }
    }
    
    return -1;  // 未找到返回-1
}

// 冒泡排序函数实现
void bubbleSort(int arr[], int size) {
    // 外层循环控制排序轮数
    for (int i = 0; i < size - 1; i++) {
        // 内层循环进行相邻元素比较和交换
        for (int j = 0; j < size - i - 1; j++) {
            if (arr[j] > arr[j + 1]) {
                // 交换相邻元素
                int temp = arr[j];
                arr[j] = arr[j + 1];
                arr[j + 1] = temp;
            }
        }
    }
}

// 选择排序函数实现
void selectionSort(int arr[], int size) {
    // 外层循环控制选择轮数
    for (int i = 0; i < size - 1; i++) {
        int minIndex = i;  // 假设当前位置是最小值
        
        // 在剩余元素中查找最小值
        for (int j = i + 1; j < size; j++) {
            if (arr[j] < arr[minIndex]) {
                minIndex = j;  // 更新最小值索引
            }
        }
        
        // 将找到的最小值交换到当前位置
        if (minIndex != i) {
            int temp = arr[i];
            arr[i] = arr[minIndex];
            arr[minIndex] = temp;
        }
    }
}

// 数组反转函数实现
void reverseArray(int arr[], int size) {
    // 使用双指针法，首尾交换
    int left = 0;
    int right = size - 1;
    
    while (left < right) {
        // 交换首尾元素
        int temp = arr[left];
        arr[left] = arr[right];
        arr[right] = temp;
        
        left++;   // 左指针右移
        right--;  // 右指针左移
    }
}

// 打印数组函数实现
void printArray(int arr[], int size) {
    printf("[");
    for (int i = 0; i < size; i++) {
        printf("%d", arr[i]);
        if (i < size - 1) {
            printf(", ");  // 元素间加分隔符
        }
    }
    printf("]\n");
}
```
