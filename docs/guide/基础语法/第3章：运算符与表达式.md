## 3.1 算术运算符（+、-、*、/、%）

算术运算符是C语言中最基础的运算符，用于执行基本的数学运算。它们包括加法（+）、减法（-）、乘法（*）、除法（/）和取模（%）。

算术运算符在所有C语言版本中都得到支持，是编程的基础工具。

### 实例方法表格

| 功能名称 | 实例调用方法 | 具体功能与注意事项                           |
| -------- | ------------ | -------------------------------------------- |
| 加法运算 | `a + b`      | 计算两个数值的和，适用于整数和浮点数         |
| 减法运算 | `a - b`      | 计算两个数值的差，适用于整数和浮点数         |
| 乘法运算 | `a * b`      | 计算两个数值的积，适用于整数和浮点数         |
| 除法运算 | `a / b`      | 计算两个数值的商，注意整数除法会截断小数部分 |
| 取模运算 | `a % b`      | 计算两个整数相除的余数，仅适用于整数类型     |

### 使用示例

```c
#include <stdio.h>

int main() {
    // 声明并初始化整数变量
    int a = 15;
    int b = 4;
    
    // 声明并初始化浮点数变量
    float x = 15.0f;
    float y = 4.0f;
    
    // 加法运算示例
    printf("加法: %d + %d = %d\n", a, b, a + b);  // 输出: 15 + 4 = 19
    
    // 减法运算示例
    printf("减法: %d - %d = %d\n", a, b, a - b);  // 输出: 15 - 4 = 11
    
    // 乘法运算示例
    printf("乘法: %d * %d = %d\n", a, b, a * b);  // 输出: 15 * 4 = 60
    
    // 整数除法示例（注意截断）
    printf("整数除法: %d / %d = %d\n", a, b, a / b);  // 输出: 15 / 4 = 3
    
    // 浮点数除法示例
    printf("浮点数除法: %.1f / %.1f = %.2f\n", x, y, x / y);  // 输出: 15.0 / 4.0 = 3.75
    
    // 取模运算示例
    printf("取模: %d %% %d = %d\n", a, b, a % b);  // 输出: 15 % 4 = 3
    
    // 错误处理：检查除零错误
    if (b != 0) {
        printf("安全的除法运算\n");
    } else {
        printf("错误：除数不能为零！\n");
    }
    
    return 0;
}
```

### 注意事项

1. **整数除法截断**：当两个整数相除时，结果会自动截断小数部分，只保留整数部分
2. **除零错误**：除法和取模运算中，除数不能为零，否则会导致运行时错误
3. **取模运算限制**：取模运算只能用于整数类型，不能用于浮点数
4. **数据类型提升**：当不同类型的数值进行运算时，会发生自动类型转换（整数提升和浮点提升）

小结：算术运算符是C语言中最基本的数学运算工具，包括加、减、乘、除和取模五种运算。掌握这些运算符的使用规则和注意事项，特别是整数除法的截断特性和取模运算的限制，对于编写正确的数学计算程序至关重要。

## 3.2 关系运算符（>、<、==、!=等）

关系运算符用于比较两个值之间的关系，返回布尔结果（在C语言中，非零值表示真，零值表示假）。这些运算符在条件判断和循环控制中扮演着重要角色。

关系运算符在所有C语言版本中都保持一致，是逻辑控制的基础。

### 实例方法表格

| 功能名称   | 实例调用方法 | 具体功能与注意事项                                   |
| ---------- | ------------ | ---------------------------------------------------- |
| 大于比较   | `a > b`      | 判断a是否大于b，返回1（真）或0（假）                 |
| 小于比较   | `a < b`      | 判断a是否小于b，返回1（真）或0（假）                 |
| 大于等于   | `a >= b`     | 判断a是否大于或等于b，返回1（真）或0（假）           |
| 小于等于   | `a <= b`     | 判断a是否小于或等于b，返回1（真）或0（假）           |
| 等于比较   | `a == b`     | 判断a是否等于b，返回1（真）或0（假），注意使用双等号 |
| 不等于比较 | `a != b`     | 判断a是否不等于b，返回1（真）或0（假）               |

### 使用示例

```c
#include <stdio.h>

int main() {
    // 声明并初始化测试变量
    int num1 = 10;
    int num2 = 20;
    float f1 = 3.14f;
    float f2 = 3.14159f;
    
    // 大于比较示例
    printf("%d > %d 的结果是: %d\n", num1, num2, num1 > num2);  // 输出: 10 > 20 的结果是: 0
    
    // 小于比较示例
    printf("%d < %d 的结果是: %d\n", num1, num2, num1 < num2);  // 输出: 10 < 20 的结果是: 1
    
    // 大于等于比较示例
    printf("%d >= %d 的结果是: %d\n", num2, num1, num2 >= num1);  // 输出: 20 >= 10 的结果是: 1
    
    // 小于等于比较示例
    printf("%d <= %d 的结果是: %d\n", num1, num2, num1 <= num2);  // 输出: 10 <= 20 的结果是: 1
    
    // 等于比较示例（注意使用双等号==，不是单等号=）
    printf("%d == %d 的结果是: %d\n", num1, num1, num1 == num1);  // 输出: 10 == 10 的结果是: 1
    
    // 不等于比较示例
    printf("%d != %d 的结果是: %d\n", num1, num2, num1 != num2);  // 输出: 10 != 20 的结果是: 1
    
    // 浮点数比较示例（注意浮点数比较的精度问题）
    printf("%.2f == %.5f 的结果是: %d\n", f1, f2, f1 == f2);  // 输出: 3.14 == 3.14159 的结果是: 0
    
    // 在if语句中的实际应用
    if (num1 < num2) {
        printf("num1确实小于num2！\n");  // 这行会被执行
    }
    
    // 错误示例：避免常见的赋值vs比较错误
    // if (num1 = num2) {  // 这是赋值，不是比较！会导致逻辑错误
    //     printf("这是错误的写法！\n");
    // }
    
    return 0;
}
```

### 注意事项

1. **等于vs赋值**：最容易犯的错误是使用单等号（=）进行比较，实际上应该使用双等号（==）
2. **浮点数比较**：由于浮点数的精度问题，直接使用==比较两个浮点数可能产生意外结果，通常应该比较它们的差值是否小于某个很小的阈值
3. **返回值类型**：关系运算符返回int类型的值（0或1），而不是布尔类型
4. **运算优先级**：关系运算符的优先级低于算术运算符，所以表达式如`a + b > c`会先计算`a + b`，再进行比较

小结：关系运算符用于比较两个值的大小关系，包括大于、小于、等于等六种运算。正确使用这些运算符对于实现条件判断和逻辑控制至关重要，特别要注意避免将赋值运算符（=）与等于比较运算符（==）混淆，以及理解浮点数比较的特殊性。

## 3.3 逻辑运算符（&&、||、!）

逻辑运算符用于组合多个条件表达式，实现复杂的逻辑判断。C语言提供了三种逻辑运算符：逻辑与（&&）、逻辑或（||）和逻辑非（!）。

逻辑运算符在所有C语言版本中都保持一致，是构建复杂逻辑表达式的基础工具。

### 实例方法表格

| 功能名称 | 实例调用方法     | 具体功能与注意事项                           |
| -------- | ---------------- | -------------------------------------------- |
| 逻辑与   | `expr1 && expr2` | 当两个表达式都为真时返回真，具有短路求值特性 |
| 逻辑或   | `expr1           |                                              |
| 逻辑非   | `!expr`          | 对表达式的结果取反，真变假，假变真           |

### 使用示例

```c
#include <stdio.h>

int main() {
    // 声明并初始化测试变量
    int age = 25;
    int hasLicense = 1;  // 1表示有驾照，0表示没有
    int score = 85;
    int isPresent = 0;   // 0表示缺席，1表示出席
    
    // 逻辑与运算示例（需要同时满足两个条件）
    printf("年龄>=18且有驾照: %d && %d = %d\n", 
           (age >= 18), hasLicense, (age >= 18) && hasLicense);
    // 输出: 年龄>=18且有驾照: 1 && 1 = 1
    
    // 逻辑或运算示例（满足任一条件即可）
    printf("分数>=90或出席: %d || %d = %d\n", 
           (score >= 90), isPresent, (score >= 90) || isPresent);
    // 输出: 分数>=90或出席: 0 || 0 = 0
    
    // 逻辑非运算示例（取反操作）
    printf("缺席的反面: !%d = %d\n", isPresent, !isPresent);
    // 输出: 缺席的反面: !0 = 1
    
    // 短路求值演示 - 逻辑与的短路特性
    int test1 = 0;
    int test2 = 1;
    // 由于test1为0（假），test2的表达式不会被计算
    if (test1 && (test2 = 0)) {
        printf("这行不会被执行\n");
    }
    printf("短路求值后test2的值: %d\n", test2);  // 输出: 短路求值后test2的值: 1
    
    // 短路求值演示 - 逻辑或的短路特性
    int test3 = 1;
    int test4 = 0;
    // 由于test3为1（真），test4的表达式不会被计算
    if (test3 || (test4 = 1)) {
        printf("这行会被执行\n");
    }
    printf("短路求值后test4的值: %d\n", test4);  // 输出: 短路求值后test4的值: 0
    
    // 复杂逻辑表达式示例
    int isValidUser = (age >= 18) && hasLicense && (score > 80);
    printf("用户是否有效: %d\n", isValidUser);  // 输出: 用户是否有效: 1
    
    // 使用逻辑非进行条件反转
    if (!isPresent) {
        printf("用户缺席，需要跟进\n");  // 这行会被执行
    }
    
    return 0;
}
```

### 注意事项

1. **短路求值**：逻辑与（&&）和逻辑或（||）都具有短路求值特性。对于`A && B`，如果A为假，则B不会被计算；对于`A || B`，如果A为真，则B不会被计算
2. **运算优先级**：逻辑非（!）的优先级最高，逻辑与（&&）次之，逻辑或（||）最低。但建议使用括号明确表达意图
3. **返回值**：逻辑运算符返回int类型的值（0或1），而不是布尔类型
4. **可读性**：复杂的逻辑表达式应该适当使用括号分组，提高代码可读性

小结：逻辑运算符包括逻辑与（&&）、逻辑或（||）和逻辑非（!），用于构建复杂的条件表达式。理解短路求值特性对于编写高效且安全的代码非常重要，特别是在涉及函数调用或可能产生副作用的表达式时。合理使用逻辑运算符可以大大简化条件判断逻辑。

## 3.4 位运算符（&、|、^、~、<<、>>）

位运算符直接对整数的二进制位进行操作，是C语言的特色功能之一。这些运算符在系统编程、嵌入式开发和性能优化中非常有用。

位运算符在所有C语言版本中都得到支持，是底层编程的重要工具。

### 实例方法表格

| 功能名称 | 实例调用方法 | 具体功能与注意事项                                           |
| -------- | ------------ | ------------------------------------------------------------ |
| 按位与   | `a & b`      | 对两个操作数的每一位进行与运算，同为1则结果为1               |
| 按位或   | `a           | b`                                                           |
| 按位异或 | `a ^ b`      | 对两个操作数的每一位进行异或运算，不同则结果为1              |
| 按位取反 | `~a`         | 对操作数的每一位进行取反操作，0变1，1变0                     |
| 左移     | `a << n`     | 将a的二进制位向左移动n位，右侧补0                            |
| 右移     | `a >> n`     | 将a的二进制位向右移动n位，左侧补符号位（有符号）或0（无符号） |

### 使用示例

```c
#include <stdio.h>

// 辅助函数：打印整数的二进制表示
void printBinary(unsigned int num, int bits) {
    for (int i = bits - 1; i >= 0; i--) {
        printf("%d", (num >> i) & 1);
        if (i % 4 == 0 && i > 0) printf(" ");  // 每4位加空格便于阅读
    }
}

int main() {
    // 声明并初始化测试变量
    unsigned char a = 12;   // 二进制: 0000 1100
    unsigned char b = 10;   // 二进制: 0000 1010
    
    printf("a = %d (二进制: ", a);
    printBinary(a, 8);
    printf(")\n");
    
    printf("b = %d (二进制: ", b);
    printBinary(b, 8);
    printf(")\n\n");
    
    // 按位与运算示例
    unsigned char andResult = a & b;
    printf("按位与 (a & b): ");
    printBinary(andResult, 8);
    printf(" = %d\n", andResult);  // 结果: 0000 1000 = 8
    
    // 按位或运算示例
    unsigned char orResult = a | b;
    printf("按位或 (a | b): ");
    printBinary(orResult, 8);
    printf(" = %d\n", orResult);   // 结果: 0000 1110 = 14
    
    // 按位异或运算示例
    unsigned char xorResult = a ^ b;
    printf("按位异或 (a ^ b): ");
    printBinary(xorResult, 8);
    printf(" = %d\n", xorResult);  // 结果: 0000 0110 = 6
    
    // 按位取反运算示例
    unsigned char notResult = ~a;
    printf("按位取反 (~a): ");
    printBinary(notResult, 8);
    printf(" = %d\n", notResult);  // 结果: 1111 0011 = 243 (对于unsigned char)
    
    // 左移运算示例
    unsigned char leftShift = a << 2;
    printf("左移2位 (a << 2): ");
    printBinary(leftShift, 8);
    printf(" = %d\n", leftShift);  // 结果: 0011 0000 = 48
    
    // 右移运算示例
    unsigned char rightShift = a >> 2;
    printf("右移2位 (a >> 2): ");
    printBinary(rightShift, 8);
    printf(" = %d\n", rightShift); // 结果: 0000 0011 = 3
    
    // 实际应用示例：位标志操作
    unsigned char flags = 0;       // 初始标志位全为0
    unsigned char READ_FLAG = 1;   // 0000 0001
    unsigned char WRITE_FLAG = 2;  // 0000 0010
    unsigned char EXEC_FLAG = 4;   // 0000 0100
    
    // 设置读写权限
    flags |= READ_FLAG | WRITE_FLAG;  // 使用按位或设置标志位
    printf("\n设置读写权限后: ");
    printBinary(flags, 8);
    printf(" = %d\n", flags);
    
    // 检查是否有读权限
    if (flags & READ_FLAG) {
        printf("具有读权限\n");
    }
    
    // 移除写权限
    flags &= ~WRITE_FLAG;  // 使用按位与和按位取反清除标志位
    printf("移除写权限后: ");
    printBinary(flags, 8);
    printf(" = %d\n", flags);
    
    // 快速乘除法示例（左移相当于乘以2的幂，右移相当于除以2的幂）
    int number = 16;
    printf("\n快速运算示例:\n");
    printf("%d * 4 = %d (使用左移: %d << 2)\n", number, number << 2, number);
    printf("%d / 4 = %d (使用右移: %d >> 2)\n", number, number >> 2, number);
    
    return 0;
}
```

### 注意事项

1. **数据类型限制**：位运算符只能用于整数类型（char、short、int、long等），不能用于浮点数
2. **符号扩展**：右移有符号整数时，左侧补符号位（算术右移）；右移无符号整数时，左侧补0（逻辑右移）
3. **移位数量**：移位的数量不能为负数，也不能大于或等于操作数的位数，否则行为未定义
4. **溢出问题**：左移可能导致高位丢失（溢出），需要注意数据范围
5. **可移植性**：不同平台的整数大小可能不同，编写可移植代码时要谨慎

小结：位运算符提供了对整数二进制位的直接操作能力，包括按位与、或、异或、取反、左移和右移。这些运算符在位标志管理、硬件编程和性能优化中有重要应用。虽然位运算功能强大，但使用时需要注意数据类型限制、符号扩展和溢出等问题，确保代码的正确性和可移植性。

## 3.5 赋值运算符与复合赋值

赋值运算符用于将值存储到变量中，而复合赋值运算符则结合了算术或位运算与赋值操作，提供更简洁的语法。

赋值运算符是C语言的基本组成部分，在所有版本中都保持一致。

### 实例方法表格

| 功能名称   | 实例调用方法 | 具体功能与注意事项                    |
| ---------- | ------------ | ------------------------------------- |
| 简单赋值   | `a = b`      | 将b的值赋给a，返回a的新值             |
| 加法赋值   | `a += b`     | 等价于`a = a + b`，先计算a+b再赋值给a |
| 减法赋值   | `a -= b`     | 等价于`a = a - b`，先计算a-b再赋值给a |
| 乘法赋值   | `a *= b`     | 等价于`a = a * b`，先计算a*b再赋值给a |
| 除法赋值   | `a /= b`     | 等价于`a = a / b`，先计算a/b再赋值给a |
| 取模赋值   | `a %= b`     | 等价于`a = a % b`，先计算a%b再赋值给a |
| 位运算赋值 | `a &= b`, `a | = b`, `a ^= b`, `a <<= n`, `a >>= n`  |

### 使用示例

```c
#include <stdio.h>

int main() {
    // 简单赋值示例
    int x = 10;        // 初始化x为10
    int y = x;         // 将x的值赋给y
    printf("简单赋值: x = %d, y = %d\n", x, y);  // 输出: x = 10, y = 10
    
    // 加法赋值示例
    int sum = 0;
    sum += 5;          // 等价于 sum = sum + 5
    sum += 10;         // 等价于 sum = sum + 10
    printf("加法赋值: sum = %d\n", sum);  // 输出: sum = 15
    
    // 减法赋值示例
    int difference = 20;
    difference -= 7;   // 等价于 difference = difference - 7
    printf("减法赋值: difference = %d\n", difference);  // 输出: difference = 13
    
    // 乘法赋值示例
    int product = 3;
    product *= 4;      // 等价于 product = product * 4
    printf("乘法赋值: product = %d\n", product);  // 输出: product = 12
    
    // 除法赋值示例
    float quotient = 25.0f;
    quotient /= 2;     // 等价于 quotient = quotient / 2
    printf("除法赋值: quotient = %.1f\n", quotient);  // 输出: quotient = 12.5
    
    // 取模赋值示例
    int remainder = 17;
    remainder %= 5;    // 等价于 remainder = remainder % 5
    printf("取模赋值: remainder = %d\n", remainder);  // 输出: remainder = 2
    
    // 位运算复合赋值示例
    unsigned char flags = 0b00001010;  // 二进制: 0000 1010
    printf("初始标志: %08b\n", flags);
    
    flags |= 0b00000100;   // 设置第3位（从右数，从0开始）
    printf("设置位后: %08b\n", flags);  // 输出: 0000 1110
    
    flags &= 0b11111101;   // 清除第1位
    printf("清除位后: %08b\n", flags);  // 输出: 0000 1100
    
    // 左移赋值示例
    int number = 8;
    number <<= 2;      // 等价于 number = number << 2
    printf("左移赋值: %d (原值8左移2位等于32)\n", number);  // 输出: 32
    
    // 右移赋值示例
    int shifted = 32;
    shifted >>= 3;     // 等价于 shifted = shifted >> 3
    printf("右移赋值: %d (原值32右移3位等于4)\n", shifted);  // 输出: 4
    
    // 赋值运算符的返回值特性（链式赋值）
    int a, b, c;
    a = b = c = 100;   // 从右到左赋值，等价于 a = (b = (c = 100))
    printf("链式赋值: a = %d, b = %d, c = %d\n", a, b, c);  // 输出: a = 100, b = 100, c = 100
    
    // 在表达式中使用赋值运算符（谨慎使用）
    int result;
    if ((result = getValue()) > 0) {  // 先赋值再比较
        printf("getValue()返回正数: %d\n", result);
    }
    
    return 0;
}

// 辅助函数：模拟获取值的函数
int getValue() {
    return 42;  // 返回一个正数用于演示
}
```

### 注意事项

1. **运算顺序**：复合赋值运算符的执行顺序是先进行运算，再进行赋值
2. **类型转换**：赋值过程中可能发生隐式类型转换，需要注意精度损失
3. **链式赋值**：赋值运算符从右到左结合，支持链式赋值
4. **表达式中的赋值**：在条件表达式中使用赋值运算符时，务必使用括号避免优先级问题
5. **副作用**：复合赋值运算符会修改左操作数的值，这在某些上下文中可能产生意外的副作用

小结：赋值运算符（=）用于将值存储到变量中，而复合赋值运算符（+=、-=、*=等）提供了更简洁的语法来执行"运算后赋值"的操作。这些运算符不仅提高了代码的可读性，还能减少重复代码。理解赋值运算符的返回值特性和结合性对于编写复杂的表达式非常重要，但也要注意避免过度使用导致代码难以理解。

## 3.6 运算符优先级与结合性

运算符优先级决定了表达式中运算符的执行顺序，而结合性则决定了相同优先级运算符的执行方向。理解这些规则对于编写正确的表达式至关重要。

运算符优先级和结合性规则在所有C语言版本中保持一致，是理解表达式求值顺序的基础。

### 实例方法表格

| 优先级级别 | 运算符                                              | 结合性         | 说明                         |
| ---------- | --------------------------------------------------- | -------------- | ---------------------------- |
| 1（最高）  | `()` `[]` `->` `.`                                  | 左到右         | 函数调用、数组下标、成员访问 |
| 2          | `!` `~` `++` `--` `+` `-` `*` `&` `(type)` `sizeof` | 右到左         | 一元运算符、类型转换、sizeof |
| 3          | `*` `/` `%`                                         | 左到右         | 乘法、除法、取模             |
| 4          | `+` `-`                                             | 左到右         | 加法、减法                   |
| 5          | `<<` `>>`                                           | 左到右         | 位移运算                     |
| 6          | `<` `<=` `>` `>=`                                   | 左到右         | 关系运算符                   |
| 7          | `==` `!=`                                           | 左到右         | 相等性运算符                 |
| 8          | `&`                                                 | 左到右         | 按位与                       |
| 9          | `^`                                                 | 左到右         | 按位异或                     |
| 10         | `                                                   | `              | 左到右                       |
| 11         | `&&`                                                | 左到右         | 逻辑与                       |
| 12         | `                                                   |                | `                            |
| 13         | `?:`                                                | 右到左         | 条件运算符                   |
| 14         | `=` `+=` `-=` `*=` `/=` `%=` `&=` `^=` `            | =` `<<=` `>>=` | 右到左                       |
| 15（最低） | `,`                                                 | 左到右         | 逗号运算符                   |

### 使用示例

```c
#include <stdio.h>

int main() {
    // 示例1：算术运算符优先级
    int result1 = 10 + 5 * 3;        // 乘法优先级高于加法
    printf("10 + 5 * 3 = %d\n", result1);  // 输出: 25 (不是45)
    
    // 使用括号改变优先级
    int result2 = (10 + 5) * 3;      // 括号内的加法先执行
    printf("(10 + 5) * 3 = %d\n", result2);  // 输出: 45
    
    // 示例2：关系运算符与逻辑运算符
    int a = 5, b = 10, c = 15;
    int result3 = a < b && b < c;    // 关系运算符优先级高于逻辑与
    printf("5 < 10 && 10 < 15 = %d\n", result3);  // 输出: 1 (真)
    
    // 示例3：赋值运算符的右结合性
    int x, y, z;
    x = y = z = 20;                  // 从右到左：x = (y = (z = 20))
    printf("x = %d, y = %d, z = %d\n", x, y, z);  // 输出: x = 20, y = 20, z = 20
    
    // 示例4：一元运算符的右结合性
    int num = 5;
    int result4 = ++num + num++;     // 先执行++num(变为6)，再执行num++(使用6后变为7)
    printf("++num + num++ = %d, num = %d\n", result4, num);  // 输出: 12, num = 7
    
    // 示例5：位运算符优先级陷阱
    int flag1 = 1, flag2 = 2, flag3 = 4;
    int combined1 = flag1 | flag2 & flag3;  // &优先级高于|，等价于 flag1 | (flag2 & flag3)
    printf("flag1 | flag2 & flag3 = %d\n", combined1);  // 输出: 1 (因为2&4=0, 1|0=1)
    
    // 正确的位标志组合（使用括号）
    int combined2 = (flag1 | flag2) & flag3;  // 明确指定运算顺序
    printf("(flag1 | flag2) & flag3 = %d\n", combined2);  // 输出: 0
    
    // 示例6：条件运算符的右结合性
    int score = 85;
    char grade = score >= 90 ? 'A' : score >= 80 ? 'B' : 'C';
    // 等价于: score >= 90 ? 'A' : (score >= 80 ? 'B' : 'C')
    printf("分数%d对应的等级: %c\n", score, grade);  // 输出: B
    
    // 示例7：逗号运算符
    int i = (1, 2, 3);               // 逗号运算符返回最后一个表达式的值
    printf("i = (1, 2, 3) = %d\n", i);  // 输出: 3
    
    // 示例8：sizeof运算符的特殊性
    int arr[10];
    size_t size1 = sizeof(arr);      // sizeof是运算符，不是函数
    size_t size2 = sizeof arr;       // 可以不用括号（当操作数是标识符时）
    printf("数组大小: %zu 字节\n", size1);  // 输出: 40字节（假设int为4字节）
    
    // 示例9：避免优先级混淆的最佳实践
    // 即使知道优先级，也建议在复杂表达式中使用括号
    int complex1 = (a + b) * (c - d) / (e + f);  // 清晰明了
    // 而不是: int complex2 = a + b * c - d / e + f;  // 难以理解
    
    return 0;
}
```

### 注意事项

1. **记忆负担**：C语言有15个优先级级别，完全记住很困难，建议在不确定时使用括号

2. 常见陷阱：

   - 位运算符（&、|、^）的优先级低于关系运算符（==、!=等）
   - 赋值运算符的优先级很低，通常需要括号
   - 逻辑运算符（&&、||）的优先级低于关系运算符
   
3. **可读性优先**：即使表达式按照优先级规则是正确的，为了代码可读性也应该适当使用括号

4. **结合性影响**：对于相同优先级的运算符，结合性决定了执行顺序（左到右或右到左）

5. **编译器警告**：现代编译器会对可疑的优先级使用发出警告，应该认真对待这些警告
