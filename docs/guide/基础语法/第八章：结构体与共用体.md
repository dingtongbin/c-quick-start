# 第八章：结构体与共用体

## 8.1 结构体的定义与使用

在 C 语言中，结构体（`struct`）是一种用户自定义的数据类型，它允许我们将不同类型的数据组合在一起，形成一个逻辑单元。想象一下，如果你要描述一个学生的信息，包括姓名、年龄、学号、成绩等，这些数据类型各不相同（字符串、整数、浮点数），而结构体就是把它们打包成一个“学生”对象的好方法。

### 版本信息表格

| 项目       | 说明                                                         |
| ---------- | ------------------------------------------------------------ |
| C 标准支持 | C89 起即支持结构体，后续标准（C99、C11、C17）增强但核心语法不变 |
| 编译器要求 | GCC 4.0+、Clang 3.0+、MSVC 2005+ 均完全支持                  |
| 使用场景   | 表示具有多个属性的实体，如学生、员工、商品、坐标点等         |

### 实例方法表格

| 功能名称       | 实例调用方法                              | 具体功能与注意事项                                           |
| -------------- | ----------------------------------------- | ------------------------------------------------------------ |
| 定义结构体     | `struct Student { ... };`                 | 使用 `struct` 关键字定义新类型，大括号内声明成员变量         |
| 声明结构体变量 | `struct Student s1;`                      | 声明一个结构体类型的变量，此时未初始化                       |
| 初始化结构体   | `struct Student s1 = {"张三", 18, 1001};` | 按成员顺序初始化，C99 起支持指定初始化（如 `.name = "张三"`） |
| 访问成员       | `s1.age = 19;`                            | 使用点运算符 `.` 访问或修改成员                              |

### 使用示例

```c
#include <stdio.h>
#include <string.h>

// 定义一个结构体类型：Student
struct Student {
    char name[20];   // 学生姓名，字符数组
    int age;         // 年龄，整型
    int id;          // 学号，整型
    float score;     // 成绩，浮点型
};

int main() {
    // 声明并初始化一个结构体变量
    struct Student s1 = {"李四", 20, 1002, 88.5};

    // 打印结构体成员
    printf("姓名: %s\n", s1.name);     // 输出：李四
    printf("年龄: %d\n", s1.age);      // 输出：20
    printf("学号: %d\n", s1.id);       // 输出：1002
    printf("成绩: %.2f\n", s1.score);  // 输出：88.50

    // 修改成员值
    s1.age = 21;
    strcpy(s1.name, "王五");  // 注意：字符串不能直接赋值，需用 strcpy

    printf("\n修改后:\n");
    printf("姓名: %s\n", s1.name);     // 输出：王五
    printf("年龄: %d\n", s1.age);      // 输出：21

    return 0;
}
```

> **注意**：字符串成员（如 `name`）是字符数组，不能用 `s1.name = "新名字"` 赋值，必须使用 `strcpy` 或初始化时指定。

### 注意事项

- 结构体定义只是“蓝图”，不占用内存；只有声明变量时才分配内存。
- 结构体成员可以是基本类型、数组，甚至其他结构体（嵌套）。
- C 语言中结构体变量可以直接整体赋值（如 `s2 = s1;`），这是合法的。
- 初始化时若未提供全部值，剩余成员会被自动初始化为 0（或空字符串）。
